高校数学の「亀裂」を辿る

この証明、
本当に正しいのか？

limx→0 (sin x / x) = 1

高校で習ったあの証明。図を使って、はさみうちで、「ほら1になった」。
でも待ってほしい。その証明の中に、静かな矛盾が潜んでいる。

Section 01

高校で習った証明

高校数学III

まず数値で確かめてみよう。x を 0 に近づけると sin x / x はどう変わるか。

確かに 1 に近づいている。では「なぜ？」を証明する。教科書の方法はこうだ。

▷ 単位円で 0 < x < π/2 のとき：
　△OAB の面積 < 扇形OAB の面積 < △OAT の面積
　(1/2)sin x < (1/2)x < (1/2)tan x

x → 0 のとき cos x → 1 なので
はさみうちの原理より　lim(sin x / x) = 1　□

美しい。スライダーで x を動かすと、3つの図形の大小関係が目で確認できる。完璧に見える。

Section 02

証明の中の亀裂

だがここで立ち止まってほしい。この証明の中に、ひとつ「当たり前に使っているもの」がある。

扇形 OAB の面積　＝　(1/2) × r² × x　＝　x/2　（r = 1 のとき）

この公式をどこで習ったか覚えているか？
「扇形の面積 = π r² × (中心角 / 2π)」から来ている。
そして π は…円の面積から来ている。

⚠ 循環の構造

扇形の面積公式　←　円の面積 πr²　を使う
円の面積 πr²　←　∫√(1-x²) dx　を使う
∫√(1-x²) dx　←　三角関数で置換積分する
置換積分で　sin の微分　が出てくる
sin の微分　←　lim(sin x / x) = 1　を使う

── つまり「証明したいもの」を前提に使っている。

高校では「扇形の面積はこういうもの」として自明に使っているが、 厳密に掘り下げると lim(sin x/x) = 1 がすでに顔を出している。

もうひとつの問題：「近づける」とは何か

「x を 0 に近づけると sin x / x は 1 に近づく」──
この「近づく」という言葉、実は定義されていない。
どのくらい近いと「近い」のか？
「限りなく近づく」とは数学的に何を意味するのか？

高校では直感で済ませているが、ここに大学数学の入口がある。

Section 03

「近づく」を数で定義する
ε-δ 論法への入口

大学数学（解析学）

「f(x) が x→a で L に収束する」とはどういう意味か。大学では次のように定義する。

定義（ε-δ論法）

任意の ε > 0 に対して、
ある δ > 0 が存在して、
0 < |x - a| < δ　ならば　|f(x) - L| < ε

このとき　limx→a f(x) = L　と書く。

言葉にすると：「どんなに小さい誤差 ε を要求されても、それに応えられる近さ δ が必ず取れる」。

下のスライダーで体験してみよう。f(x) = sin x / x、a = 0、L = 1 の場合だ。

0.10
許容する誤差（どのくらい 1 に近ければよいか）
0.50
x の近さ（0 からこの距離以内）
確認中...
赤帯：|f(x) - 1| < ε の範囲　／　青帯：|x| < δ の範囲
青帯内のグラフが完全に赤帯の中に収まれば「このε に対してこの δ が使える」。

δ を小さくしていくと、どんな ε にも対応できることが目で確認できる。 これが「収束」の厳密な意味だ。

✓ 循環を回避した証明（大学版）

大学の解析学では sin x を最初からべき級数（テイラー展開）で定義する：
sin x = x - x³/6 + x⁵/120 - …

このとき sin x / x = 1 - x²/6 + x⁴/120 - …
→ x → 0 で 1 に収束することは、ε-δ で厳密に示せる。

扇形の面積も、円周率も、一切使わない。循環は消える。

Section 04

高校から大学数学へ

「高校の証明は間違っていたのか？」

間違いではない。直感的に正しく、実用上も十分だ。
ただ「なぜ正しいか」を突き詰めようとすると、言葉の定義が必要になる。

「近づく」とはどういうことか。
この問いに答えようとした先に、ε-δ論法がある。
そして解析学の世界が開く。

数学は高校で完成しているわけではない。むしろ高校数学は、 より深い問いへの入口として設計されている。 「なぜ？」と感じた瞬間が、大学数学の始まりだ。
