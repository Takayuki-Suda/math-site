🦅 放物線の渓谷
y = a(x − p)² + q

STAGE 4 ── パラボラファルコンは放物線を描いて飛ぶ。a・p・q を動かし、軌道を支配しろ。

頂点
(0, 0)
軸
x = 0
開き方
上に開く
y = 1(x − 0)² + 0
a 放物線の開き方・向き（正:上向き、負:下向き） 1
p 頂点の x 座標（左右の移動） 0
q 頂点の y 座標（上下の移動） 0
二次関数とは、x の2乗が含まれる関数です。グラフは放物線（U字型の曲線）になります。

y = a(x − p)² + q という形で表すと、頂点・軸・開き方がひと目でわかります。

ボールを投げたときの軌跡や、橋のアーチの形など、自然界にも多く現れます。
a（係数）は放物線の「開き方」と「向き」を決めます。

・a > 0 → 上に開く（谷型）
・a < 0 → 下に開く（山型）
・|a| が大きい → 細くとがった放物線
・|a| が小さい → 横に広がった放物線

a だけを動かして確かめてみよう！p と q は 0 のまま。
p と q は放物線の位置を決めます。

・p を変える → 放物線が左右に移動（形は変わらない）
・q を変える → 放物線が上下に移動（形は変わらない）

注意：p = 2 のとき式は y = a(x − 2)² + q。符号に気をつけよう！

a を固定して p・q だけ動かすと、形はそのままで平行移動するのが確認できます。
頂点とは放物線の「てっぺん（または底）」の点です。

y = a(x − p)² + q の頂点は (p, q) です。

軸とは放物線の左右対称の直線で、x = p になります。

上の「頂点」「軸」の表示がリアルタイムで変わるのを確認しよう！
