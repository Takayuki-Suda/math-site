行間ノート ─ 弱Mordell–Weil N1/N3

Kummer 対の完全証明
The Kummer Pairing, with every step justified

完全対 E(K)/mE(K) × GL/K → E[m] の構成を、well-defined 性・双線形性・両側の核の決定・ 商への降下・2つの単射の導出まで、一行も飛ばさずに実行します。 「E[m] ⊂ E(K) がどこで効くか」の完全な一覧表付き。

TEXT: Silverman, AEC 2nd ed. ─ VIII.1.2（完全展開）

本編Ⅱ 弱MW N0 帰着補題 N1 Kummer対 N2 不分岐性 N3 体論の有限性
Section 01

このノートで埋める行間

  • ① [m] : E(K) → E(K) の全射性の証明（本編ではチップ引用のみ）
  • ②「別の選択は Q + T（T ∈ E[m]）の形しかない」の理由
  • ③ 合成 στ の作用規約（Pστ = (Pσ)τ）の明示──これを曖昧にするとコサイクル計算が検証不能になる
  • ④ 左核の決定で使う「無限次 Galois 拡大の固定点」の扱い
  • ⑤ 右核の決定における「選ばれた Q での値が O」から「全ての Q が固定される」への渡り
  • ⑥ κ を商 E(K)/mE(K) × GL/K に降ろす手続き（両側それぞれ確認が要る）
  • ⑦ 完全対から2つの単射 δ, ρ を得る一般論と、「有限 ⟺ 有限」の両方向の証明

規約：このノート全体で E[m] ⊂ E(K) を仮定します（N0 の系で正当化済み）。 G := GK/K と略記します。 N0 の前提 P1（群構造）・P2（作用と加法の可換性）・P3（固定点）・P4（#E[m] = m²）は引き続き使用します。

Section 02

追加の前提（Q1–Q4）

前提 Q1 ─ [m] の全射性

[m] : E(K) → E(K) は全射である。

証明を開く（行間①）

[m] は次数 m² の isogenyAEC III.6.2(a)、特に非定数な曲線の射である。 滑らかな射影曲線の間の非定数射は、代数閉体上の点集合の上で全射AEC II.2.3。 よって任意の P ∈ E(K) に対し [m]Q = P なる Q ∈ E(K) が存在する。 ここで引用した2つは AEC 第II・III章で証明済みの定理であり、この2つの引用が「割り算はいつでも K の中でできる」ことの全根拠である。

前提 Q2 ─ 絶対Galois群の固定点

E(K)G = E(K)。根拠：N0 の P3 と同じ議論を M = K, F = K に適用する （KG = K は char 0 の体論。K/K は Galois 拡大である）。

前提 Q3 ─ 作用の合成規約（行間③）

指数表記 Pσ に対して、合成は右作用の規約で読む：

Pστ := (Pσ)τ

以後のコサイクル型の計算はすべてこの規約の下で検証される（AEC の指数表記と同じ規約）。

前提 Q4 ─ 制限写像の全射性（体論）

M/K を K 内の（有限次または無限次の）Galois 拡大とすると、制限写像 res : G → GM/K, σ ↦ σ|M は全射で、核は GK/M。 よって G/GK/M ≅ GM/K。 また Galois 拡大では（無限次でも）[M:K] = |GM/K|（基数として）。 根拠：同型延長定理（M の K-自己同型は K の自己同型に延長できる）。

Section 03 ─ (a)

定義と well-defined 性

定義（Kummer対） AEC VIII §1

P ∈ E(K), σ ∈ G に対し、[m]Q = P なる Q ∈ E(K) を選び（存在：Q1）

κ(P, σ) := Qσ − Q
1

値は E[m] に入る：κ(P,σ) ∈ E[m]。

▸ 根拠　[m](Qσ − Q) = [m](Qσ) − [m]Q（P1）= ([m]Q)σ − [m]Q（P2）= Pσ − P = O（P ∈ E(K)、P2）。

2

Q の別の選択は Q + T（T ∈ E[m]）の形に限る。

▸ 根拠　[m]Q′ = P = [m]Q なら [m](Q′ − Q) = O（P1）、つまり T := Q′ − Q ∈ E[m] = ker[m]。これが行間②：「差を取れば核に落ちる」という核の定義の適用である。

3

値は選択に依らない：(Q+T)σ − (Q+T) = Qσ − Q。

▸ 根拠　(Q+T)σ = Qσ + Tσ（P2）。整理すると差は Tσ − T。T ∈ E[m] ⊂ E(K)（仮定）より Tσ = T（P2）なので差は O。──E[m] ⊂ E(K) の使用箇所【その1】。

(a) ∎
Section 04 ─ (b)

双線形性

P についての線形性

4

P₁, P₂ ∈ E(K) に対し Q₁, Q₂ を選ぶと、Q₁ + Q₂ は P₁ + P₂ の合法な選択である。

▸ 根拠　[m](Q₁+Q₂) = [m]Q₁ + [m]Q₂ = P₁ + P₂（P1）。

5

κ(P₁+P₂, σ) = κ(P₁,σ) + κ(P₂,σ)。

▸ 根拠　(a) で値は選択に依らないと示したので、Step 4 の選択で計算してよい：(Q₁+Q₂)σ − (Q₁+Q₂) = (Q₁σ − Q₁) + (Q₂σ − Q₂)（P2 とアーベル性）。well-defined 性を先に証明しておくことが、この1行の論理的前提になっている点に注意。

σ についての線形性

6

κ(P, στ) = Qστ − Q = (Qστ − Qτ) + (Qτ − Q)。

▸ 根拠　中間項 ±Qτ の挿入（アーベル群での恒等式）。

7

Qστ − Qτ = (Qσ − Q)τ = κ(P,σ)τ。

▸ 根拠　Qστ = (Qσ)τ（Q3 の規約──ここで規約が本当に使われる）と (A−B)τ = Aτ − Bτ（P2）。

8

κ(P,σ)τ = κ(P,σ)。したがって κ(P,στ) = κ(P,σ) + κ(P,τ)。

▸ 根拠　κ(P,σ) ∈ E[m] ⊂ E(K)（Step 1 と仮定）なので τ で不動（P2）。──E[m] ⊂ E(K) の使用箇所【その2】。この仮定なしでは κ(P,στ) = κ(P,σ)τ + κ(P,τ) という1-コサイクル関係式止まりであり、準同型にはならない。

(b) ∎
Section 05 ─ (c)

左核 = mE(K)

9

（⊇）P ∈ mE(K) ならば κ(P,σ) = O（∀σ）。

▸ 根拠　P = [m]Q₀, Q₀ ∈ E(K) と書ける。(a) より選択 Q = Q₀ で計算してよく、Q₀σ = Q₀（P2）なので κ(P,σ) = Q₀σ − Q₀ = O。

10

（⊆）すべての σ ∈ G で κ(P,σ) = O ならば P ∈ mE(K)。

▸ 根拠　Q を1つ固定する。仮定より Qσ = Q が全ての σ ∈ G で成立、つまり Q ∈ E(K)G = E(K)（Q2──行間④：ここは絶対 Galois 群の固定点なので、無限次 Galois 拡大 K/K の固定体が K であるという体論的事実が入力である）。よって P = [m]Q ∈ mE(K)。

(c) ∎
Section 06 ─ (d)

右核 = GK/L と、L/K の Galois 性

定義

[m]−1E(K) := { Q ∈ E(K) : [m]Q ∈ E(K) }、 L := K( x(Q), y(Q) : Q ∈ [m]−1E(K), Q ≠ O )。

この時点で L/K が有限次かどうかは未知であることに注意（それを知ることこそ定理の核心）。

11

（⊇）σ ∈ GK/L ならば κ(P,σ) = O（∀P ∈ E(K)）。

▸ 根拠　P の任意の合法な選択 Q は定義により [m]−1E(K) の元なので、その座標は L に入る。σ は L を固定するから Qσ = Q。

12

（⊆）すべての P ∈ E(K) で κ(P,σ) = O ならば、σ はすべての Q ∈ [m]−1E(K) を固定する。

▸ 根拠　任意の Q ∈ [m]−1E(K) を取り P := [m]Q ∈ E(K) とおくと、この Q は κ(P,σ) の合法な選択。(a) より κ の値は選択に依らないので O = κ(P,σ) = Qσ − Q。これが行間⑤：「仮定は選ばれた Q についての式だが、well-defined 性が全ての Q への橋を架ける」。

13

したがって σ は L の生成元をすべて固定し、σ ∈ GK/L。

▸ 根拠　体の自己同型が生成元を固定すれば、生成される体全体を固定する（体論：固定される元の全体は部分体をなす）。

14

L/K は Galois 拡大である（有限次とは限らない）。

▸ 根拠　正規性：任意の σ ∈ G について、Q ∈ [m]−1E(K) ⇒ [m](Qσ) = ([m]Q)σ ∈ E(K)（P2 と、E(K) の点は σ で固定）なので集合 [m]−1E(K) は G-安定。ゆえに σ は L の生成元集合を保ち σ(L) = L。K への任意の K-埋め込みは G の元の制限（Q4 の延長定理）なので、全ての埋め込みで像が L、すなわち正規。分離性は char 0。

(d) ∎
Section 07

商への降下と、2つの単射（行間⑥⑦）

15

左側の降下：κ は E(K)/mE(K) × G 上の対 κ̄ を誘導する。

▸ 根拠　P ≡ P′ (mod mE(K)) なら P′ = P + [m]R (R ∈ E(K)) で、κ(P′,σ) = κ(P,σ) + κ([m]R,σ)（(b) の P-線形性）= κ(P,σ) + O（(c)⊇）。つまり κ(·,σ) は剰余類の上で定数。

16

右側の降下：κ(P,σ) は σ|L だけで決まる。したがって κ̄ は E(K)/mE(K) × GL/K 上の双線形対に降りる。

▸ 根拠　σ|L = σ′|L とすると σ′ = σι, ι := σ−1σ′ ∈ GK/L。(b) の σ-線形性より κ(P,σι) = κ(P,σ) + κ(P,ι) = κ(P,σ) + O（(d)⊇）。G/GK/L ≅ GL/K は Q4（(d) で L/K は Galois と示したので商が意味を持つ）。

ここまでの成果（完全対）

双線形な対

κ̄ : E(K)/mE(K) × GL/K → E[m]

で、左核（(c)）も右核（(d)）も自明なものが得られた。これが「完全対」の定義の検証である。

17

単射 δ：δ : E(K)/mE(K) ↪ Hom(GL/K, E[m])、δ(P̄) = κ̄(P̄, ·)。

▸ 根拠　各 δ(P̄) が群準同型であること＝(b) の σ-線形性。δ 自身が準同型であること＝(b) の P-線形性。単射性：δ(P̄) が零写像なら P は左核の元、(c) より P ∈ mE(K)、つまり P̄ = 0。

18

単射 ρ：ρ : GL/K ↪ Hom(E(K)/mE(K), E[m])、ρ(σ̄) = κ̄(·, σ̄)。

▸ 根拠　Step 17 と対称の議論（右核の自明性 (d) を使う）。

19

有限性の翻訳（⇐）：[L:K] < ∞ ならば E(K)/mE(K) は有限で、|E(K)/mE(K)| ≤ (m²)[L:K]。

▸ 根拠　Hom(GL/K, E[m]) ⊆ Map(GL/K, E[m])、後者の濃度は (m²)|GL/K|（P4 と Q4 の |GL/K| = [L:K]）。δ 単射（Step 17）。

20

有限性の翻訳（⇒）：E(K)/mE(K) が有限ならば [L:K] < ∞。

▸ 根拠　Hom(E(K)/mE(K), E[m]) は有限集合（有限集合から有限集合への写像の一部）。ρ 単射（Step 18）より GL/K は有限。Q4 の最後の主張（Galois 拡大では [L:K] = |GL/K|）より [L:K] < ∞。

∎
▸ 定理の証明における使われ方

弱MW定理の証明で実際に使うのは Step 19 の方向だけです： 「L/K が有限次」さえ示せば勝ち。その「有限次」を示すのが N2（不分岐性）と N3（体論の有限性）です。 Step 20 は逆向きの安心材料（翻訳が等価であること）で、本編の「⟺」の右向きに当たります。

Section 08

E[m] ⊂ E(K) はどこで効いたか ─ 完全な一覧

この仮定の使用箇所を全て数え上げておきます。第3部（コホモロジー）で仮定を外すとき、 どこが「壊れて」どこが「生き残る」かが一目で分かります。

主張E[m] ⊂ E(K) を使うか使う箇所
(a) well-defined 性使う【その1】Step 3：Tσ = T
(b) P-線形性使わない─（(a) 経由の間接依存のみ）
(b) σ-線形性使う【その2】Step 8：κ(P,σ)τ = κ(P,σ)
(c) 左核使わない─
(d) 右核・L の Galois 性使わない─
N2 の不分岐性 (VIII.1.5b)使わない─（第5部で再利用される理由）
? 仮定を外すと何が残るか（第3部の予告）

使用箇所【その1】が消えると κ(P,·) は「写像」としては定まらず、Q の取替えでコバウンダリ分だけずれます。 つまりコホモロジー類としてだけ well-defined になり、δ は E(K)/mE(K) ↪ H¹(G, E[m]) という connecting map に昇格します。 使用箇所【その2】が消えると値の群 E[m] への G-作用が非自明になり、Hom は H¹ に置き換わります。 いま手で追った各ステップが、第3部では完全列の一般論に吸収されます。
