Kummer 対を
コホモロジーで読み直す
The Kummer Pairing via Cohomology

第2部の「素手の証明」は、Galoisコホモロジーという整理棚に載せると一枚の完全列になります。 この言い換え自体は弱MW定理には必須ではありません──しかし、 Selmer群とШ群（第5部）はこの言葉でしか書けません。ここが Chapter X への入口です。

TEXT: Silverman, AEC 2nd ed. ─ Ch. VIII §2（精読）＋ Appendix B

Ⅰ 全体地図 Ⅱ 弱Mordell–Weil Ⅲ Kummer対 Ⅳ 降下法 Ⅴ Selmer群とШ 終 その先へ
Section 01

なぜ言い換えるのか

第2部で見た議論には、実は「既視感のある形」が繰り返し現れていました。 λP(σ) = Qσ − Q という写像、κ(P,στ) = κ(P,σ)τ + κ(P,τ) という計算規則、 「Q の取り替えで σ ↦ Tσ − T のズレが出る」という現象。 これらはすべて群コホモロジーの標準語彙（1-コサイクル、コバウンダリ）です。

この言い換えで得られるものは3つあります。

  • 仮定 E[m] ⊂ E(K) が不要になる：一般の E/K で単射 E(K)/mE(K) ↪ H¹(GK/K, E[m]) が定義できる。
  • 「何が障害か」が見える：楕円曲線では H¹(G, E) ≠ 0 であり、 このゼロでない群こそ第5部の主役（Ш の住処）になる。
  • 局所と大域を比較する枠組み：コホモロジーは体の取り替え（K → Kv）と相性がよく、 「局所条件で大域を切る」＝Selmer群の定義がそのまま書ける。
⏩ 必要な道具は H⁰ と H¹ だけ

この教材で使うコホモロジーは H⁰ と H¹ のみです（AECも Appendix B で H⁰・H¹ しか扱いません）。 導来関手・スペクトル系列などの一般論は一切不要。次節の「速成コース」で全部足ります。

Section 02

群コホモロジー速成 ─ H⁰ と H¹ AEC App. B §1–§2

G を群、M を G が（左から）作用するアーベル群（G-加群）とします。

定義（H⁰・1-コサイクル・H¹）
H⁰(G, M) = MG = { m ∈ M : mσ = m （∀σ） }　（不変部分）

写像 c : G → M（σ ↦ cσ）が 1-コサイクルであるとは：

cστ = cστ + cτ　（コサイクル条件）

ある m ∈ M で cσ = mσ − m と書けるものを1-コバウンダリという。そして：

H¹(G, M) = （1-コサイクル）/（1-コバウンダリ）

2つ、すぐ確かめられる基本事実があります（練習問題 1・2 でもあります）。

  • G の作用が自明なら コサイクル条件は cστ = cσ + cτ、 コバウンダリは 0 だけ。よって H¹(G, M) = Hom(G, M)（群準同型全体）。
  • G = GK/K のような射有限群では、コサイクルは連続なもの （＝ある有限次Galois拡大を経由するもの）だけを考えます。 以後この規約は暗黙としますAEC App. B §2。

短完全列 → 長完全列（低次部分）

G-加群の短完全列 0 → A → B → C → 0 から、次の完全列が生えます：

0 → AG → BG → CG →δ H¹(G,A) → H¹(G,B) → H¹(G,C)

要となるのは連結準同型 δ の作り方です。c ∈ CG を取り、 B での持ち上げ b（bの像 = c）を1つ選ぶ。すると σ ↦ bσ − b は A に値を取る1-コサイクルになり （像が C で cσ − c = 0 だから）、その類が δ(c)。持ち上げ b の取り替えは コバウンダリの差しか生まないので、類は well-defined です。

? δ は「割り算の障害」を測る

δ(c) = 0 ⟺ b を A の元で修正して G-不変にできる ⟺ c は BG から来る。 つまり δ は「不変元 c を、不変なまま B に持ち上げられるか」の障害です。 『元は存在するのに、Galois不変な元が存在しない』という食い違いを測る装置── これがコホモロジーの数論での役割のすべてと言ってよいです。

Section 03

楕円曲線の Kummer 完全列

m ≥ 2 を固定し、G = GK/K とします。 [m] : E(K) → E(K) は全射（Kは代数閉、[m]は非定数な同種写像）で、 核は E[m]。つまり G-加群の短完全列

0 → E[m] → E(K) →[m] E(K) → 0

があります。長完全列を取るとAEC VIII §2：

0 → E(K)[m] → E(K) →[m] E(K) →δ H¹(G, E[m]) → H¹(G, E(K)) →[m] H¹(G, E(K))

真ん中を切り出すと、楕円曲線の Kummer 完全列が得られます：

Kummer完全列 AEC VIII §2
0 → E(K)/mE(K) →δ H¹(G, E[m]) → H¹(G, E(K))[m] → 0

しかも δ の cocycle 表示は、[m]Q = P となる Q を選んで

δ(P) = [ σ ↦ Qσ − Q ] = [ σ ↦ κ(P, σ) ]

すなわち連結準同型 δ は第2部の Kummer 対そのものである。

前節の δ の一般構成を、B = E(K)、C = E(K)、A = E[m] に当てはめただけです。 「P を m で割ろうとしたときの Galois的なねじれ」という第2部の直観が、 「割り算の障害を測る連結準同型」という一般論に正確に一致しました。

そして第2部の命題 VIII.1.2 の (a)(b)(c) は、コホモロジーの一般論から自動的に出ます：

  • (a) well-defined 性 → δ の well-defined 性（コバウンダリの差は類に影響しない）。 E[m] ⊂ E(K) の仮定はもう不要です。
  • (b) 双線形性 → δ が群準同型であること＋コサイクル条件。
  • (c) 左核 = mE(K) → 完全列の完全性（δ の単射性）。

さらに E[m] ⊂ E(K) のときは G の E[m] への作用が自明なので H¹(G, E[m]) = Hom(G, E[m]) となり、第2部の E(K)/mE(K) ↪ Hom(GK/K, E[m]) がそのまま回収されます。

Section 04

体の Kummer 理論と Hilbert 90 ─ 決定的な違いはどこか

まったく同じ手続きを乗法群でやってみます。z ↦ zm の完全列

1 → μm → K× →m乗 K× → 1

から長完全列を取り、切り出すと：

1 → K×/(K×)m →δ H¹(G, μm) → H¹(G, K×)[m] → 0

ここで古典体論の金字塔が効きます。

定理（Hilbertの定理90） AEC B.2.5
H¹( GK/K, K× ) = 0

したがって連結準同型が同型になりますAEC VIII.2.2：

δ : K×/(K×)m ≅ H¹(G, μm),　δ(a) = [ σ ↦ ασ/α ]　（αm = a）
乗法群 K×楕円曲線 E
完全列の核μmE[m]
δ の行き先H¹(G, μm)H¹(G, E[m])
右端の障害群H¹(G, K×) = 0（Hilbert 90）H¹(G, E(K)) ≠ 0（一般に）
帰結δ は同型。理論は「完結」するδ は単射どまり。余りが残る
? この表の右下が、第5部の存在理由

乗法群では Hilbert 90 が障害を消してくれるので、「m乗根の理論」は K×/(K×)m で完結します。 楕円曲線では H¹(G, E(K)) が消えない。 この群は後に Weil–Châtelet 群と呼ばれ、その元は「E の主等質空間（torsor）」という幾何的実体を持ちます。 その中の「どこでも局所的に自明な部分」が Ш（Tate–Shafarevich群）です。 つまり第5部の主役たちは、この表の「≠ 0」の一マスから全部生まれてきます。

Section 05

帰着補題の正体 ─ inflation–restriction

第2部STEP 0の帰着補題（VIII.1.1.1）も、コホモロジーの一般論の特殊ケースでした。 使うのは inflation–restriction 完全列です。

定理（inflation–restriction） AEC B.2.4

N ⊴ G を正規部分群、M を G-加群とすると、次は完全：

0 → H¹(G/N, MN) →inf H¹(G, M) →res H¹(N, M)

G = GK/K、N = GK/L（L/K は有限次Galois、E[m] ⊂ E(L) とする）、 M = E[m] とすると G/N = GL/K で、次の可換図式ができます：

図1：帰着補題の正体。上段は第2部の完全列、下段が inflation–restriction。 縦矢印はKummer完全列由来の単射。左端の縦矢印 λ : P ↦ [λP] がまさに第2部で手作りした写像。

GL/K と E[m] は有限群なので H¹(GL/K, E[m]) は有限集合 （コサイクルは有限集合から有限集合への写像だから）。 Φ はそこに単射で入るので有限──帰着補題の証明が3行で終わります。 第2部で「手計算」した λP の単射性は、この図式の可換性＋inf の単射性の言い換えだったのです。

⏩ inf–res の証明は後回しでよい

inflation–restriction の証明自体は形式的なコサイクル計算です（練習問題 4 で扱います）。 初読では「有限次拡大との比較はこの完全列でやる」という使い方だけ覚えて進んで構いません。

Section 06

不分岐コホモロジー ─ 弱MW定理の圧縮形

第2部STEP 2の「不分岐性」も、コホモロジーの言葉に翻訳できます。

定義（不分岐なコホモロジー類） AEC VIII §2

v を K の有限素点、Iv ⊂ GK/K を（vの延長を1つ固定して得る）惰性群とする。 ξ ∈ H¹(GK/K, M) が v で不分岐とは、制限が消えること：

resIv(ξ) = 0 ∈ H¹(Iv, M)

（延長の取り方に依らないAEC X.4.1.1 / 演習B.6。）

命題 AEC VIII.2.1

S = {悪い還元} ∪ {v(m) ≠ 0} ∪ MK∞ とする。 δ : E(K)/mE(K) → H¹(G, E[m]) の像は、S の外のすべての素点で不分岐な類からなる。

証明

δ(P) の代表コサイクルは cσ = Qσ − Q（[m]Q = P）。 第2部の VIII.1.5b の証明は「v ∉ S なら惰性群 Iv の元 σ が Q を固定する」ことを示していた （しかもその議論は E[m] ⊂ E(K) を使っていない）。 よって σ ∈ Iv に対し cσ = O、つまり c の Iv への制限はコサイクルとして 0。

∎

これで弱Mordell–Weil定理は次の一行に圧縮されます：

E(K)/mE(K) ↪ H¹(GK/K, E[m]; S) := { ξ : S の外で不分岐 }
そして右辺は有限（第5部で正式に証明する X.4.3。中身は第2部STEP 3の議論そのもの）
? 「局所条件で大域の群を切る」── Selmer 群の原型

H¹(G, E[m]) は巨大な（無限の）群ですが、「各素点での振る舞い」に条件を課すと有限になる。 この「大域コホモロジー群 ∩ 局所条件」という切り方こそ、Selmer群の定義の型です。 第5部では「不分岐」よりも精密な局所条件（各 Kv で torsor が自明）で切ります。 型は今日ここで覚えてしまってください。

Section 07

E[2] ⊂ E(K) なら全部が平方類で書ける ─ 第5部への布石

コホモロジーの言い換えのご利益を、いちばん計算的な形で見ておきます。 E[2] ⊂ E(K)、すなわち

E : y² = (x − e₁)(x − e₂)(x − e₃),　e₁, e₂, e₃ ∈ K
T₁ = (e₁,0), T₂ = (e₂,0), T₃ = (e₃,0) が2分点

とします。基底 T₁, T₂ を選ぶと E[2] ≅ μ₂ × μ₂（G-加群として。μ₂ = {±1} は作用自明なので単に (ℤ/2)²）。 すると Hilbert 90 の同型 K×/(K×)² ≅ H¹(G, μ₂) により：

H¹(G, E[2]) ≅ K×/(K×)² × K×/(K×)²

つまり δ は「点 P を平方類のペアに対応させる写像」になります。その正体は驚くほど具体的です。

完全 2-降下写像 AEC X.1.4（規則の出典）/ X.1.1

単射準同型 E(K)/2E(K) ↪ K×/(K×)² × K×/(K×)² が

P = (x, y) ↦ ( x − e₁,　x − e₂ )　（x ≠ e₁, e₂ のとき）
(e₁, 0) ↦ ( (e₁−e₃)/(e₁−e₂),　e₁−e₂ ),　(e₂, 0) ↦ ( e₂−e₁,　(e₂−e₃)/(e₂−e₁) )
O ↦ (1, 1)

で与えられる。

なぜ x − ei なのか：関数 x − ei の因子は 2(Ti) − 2(O) で、 「Ti に対応する µ₂ 成分への射影」を実現する関数がちょうどこれだからです （厳密な導出は Weil 対を経由しますAEC X.1.1 とその証明。 初読では「割り算のねじれの μ₂ 成分 ＝ x − ei の平方類」と受け取って進んで構いません）。

実例：E : y² = x³ − x = x(x−1)(x+1)（第2部の曲線ふたたび）

e₁ = 0, e₂ = 1, e₃ = −1 とします。E(ℚ) = {O, (0,0), (1,0), (−1,0)} の像を計算すると：

P(b₁, b₂) = 像（平方類）計算根拠
O(1, 1)規則
(0,0) = (e₁,0)(−1, −1)(e₁−e₃)/(e₁−e₂) = 1/(−1),　e₁−e₂ = −1
(1,0) = (e₂,0)(1, 2)e₂−e₁ = 1,　(e₂−e₃)/(e₂−e₁) = 2
(−1,0)(−1, −2)x−e₁ = −1,　x−e₂ = −2

準同型性の検算：群では (0,0) + (1,0) = (−1,0)。像では (−1,−1)·(1,2) = (−1,−2)。✓

そして第2部で計算した体 L = ℚ([2]−1E(ℚ)) = ℚ(√−1, √2) を思い出してください。 いま出てきた平方類 −1, 2, −2 の平方根を添加した体が、まさに ℚ(√−1, √2) です。 「halving点の座標の体」と「δの像の平方類」は同じ情報の2つの顔── 素手の計算とコホモロジーが、ここで完全に噛み合います。

⏩ ここまでで第4部に進める

降下法（第4部）はコホモロジーを使いません。VIII §2 の内容が消化しきれなくても、 「δ = Kummer対」「H¹(G,E) ≠ 0 が後の主役」の2点だけ持って第4部へ進めます。 第5部に入る前にこのページへ戻ってくれば十分です。

Exercises

練習問題

問 1★ 基礎

[m]Q = P（P ∈ E(K)）とするとき、cσ = Qσ − Q がコサイクル条件 cστ = cστ + cτ を満たすことを確かめよ。 さらに、Q を Q + T（T ∈ E[m]）に取り替えたとき c が変わる量が コバウンダリ σ ↦ Tσ − T であることを確かめよ。

解答を開く

cστ = Qστ − Q = (Qσ − Q)τ + (Qτ − Q) = cστ + cτ。 （中央の等号は Qστ = (Qσ)τ と項の足し引き。） 取り替えについては (Q+T)σ − (Q+T) = cσ + (Tσ − T)。 σ ↦ Tσ − T は定義通り T が定めるコバウンダリなので、コホモロジー類は不変。 第2部・問1の答えのコホモロジー版である。

問 2★ 基礎

G の M への作用が自明のとき、H¹(G, M) = Hom(G, M) であることを証明せよ。

解答を開く

作用が自明なら cστ = cσ なので、コサイクル条件は cστ = cσ + cτ、すなわち c が群準同型 G → M であることに一致。 コバウンダリは cσ = mσ − m = m − m = 0 のみ。 よって H¹ = （準同型全体）/（0） = Hom(G, M)。

問 3★★ 標準

Hilbert 90（H¹(G, K×) = 0）を認めて、同型 δ : K×/(K×)m → H¹(G, μm)、 δ(a) = [σ ↦ ασ/α]（αm = a）を完全列から導けAEC VIII.2.2。 また、この同型の下で「a の類が自明 ⟺ a が m 乗数」を直接確かめよ。

解答の方針を開く

1 → μm → K× → K× → 1 の長完全列は K× →m乗 K× →δ H¹(G,μm) → H¹(G,K×) = 0。 右端が消えるので δ は全射、核は (K×)m、よって同型。 δ の式は連結準同型の一般構成（持ち上げ α、σ ↦ ασ/α ∈ μm）そのもの。 直接検証：δ(a) がコバウンダリ ⟺ ある ζ 写像… ⟺ ασ/α = βσ/β となる β ∈ K× が存在 ⟺ α/β が G-不変 ⟺ α/β ∈ K×。このとき a = αm = (α/β)mβm で、 βm は… β の類を調整すれば a ∈ (K×)m が出る（丁寧に書くのが練習）。

問 4★★ 標準

inflation–restriction 完全列（B.2.4）の完全性を、コサイクルの計算で証明せよ。 示すべきは (i) inf の単射性、(ii) res∘inf = 0、(iii) ker(res) ⊂ im(inf) の3点である。 （N ⊴ G、M は G-加群、inf は G/N-コサイクルを G に引き戻す写像、res は N への制限。）

解答の方針を開く

(i) c̄ : G/N → MN のコサイクルの引き戻し c が G 上でコバウンダリ、 つまり cσ = mσ − m とする。σ ∈ N では cσ = 0 なので mσ = m（∀σ∈N）、 よって m ∈ MN。すると c̄ は G/N 上のコバウンダリ σ̄ ↦ mσ̄ − m になる。

(ii) inf した類の N への制限は cσ（σ ∈ N）＝ c̄ の N での値 ＝ c̄ē = 0。 （コサイクルは単位元で 0：ce = ce·e = cee + ce より。）

(iii) res(c) = 0 とすると、N 上で cσ = mσ − m（σ ∈ N）と書ける。 c をコバウンダリ σ ↦ mσ − m で修正して、N 上 0 のコサイクル c′ にする。 コサイクル条件 c′στ = c′στ + c′τ で τ ∈ N とすると c′στ = c′στ：c′ は N の剰余類ごとに「ほぼ定数」。 さらに σ ∈ N, τ ∈ G の順で使うと値が MN に入ることが出て、 c′ が G/N → MN のコサイクルの引き戻しであることが確かめられる。

問 5★★★ 発展

E : y² = x(x−1)(x+1)、K = ℚ とする。本文の表（2-降下写像の像）を用いて次に答えよ。

(1) 像の4元 {(1,1), (−1,−1), (1,2), (−1,−2)} が (ℚ×/(ℚ×)²)² の部分群をなすことを確かめよ。
(2) E(ℚ) の階数が 0（E(ℚ) = 上記4点）であることを認めると、 E(ℚ)/2E(ℚ) ≅ (ℤ/2)² となる。単射 E(ℚ)/2E(ℚ) ↪ Hom(GL/ℚ, E[2]) （L = ℚ(i,√2)、第2部）の右辺の位数を求め、この単射が全射でないことを確認せよ。
(3) (2) の「余り」（像に入らない Hom の元）は何を意味するか。 第5部で学ぶ言葉（Selmer群・Ш）を先取りして予想を述べよ。

解答の方針を開く

(1) 平方類の積で閉じていることを確認するだけ：(−1,−1)·(1,2) = (−1,−2)、 (−1,−1)² = (1,1) など。位数4のKleinの四元群。

(2) GL/ℚ ≅ (ℤ/2)²、E[2] ≅ (ℤ/2)² なので Hom((ℤ/2)², (ℤ/2)²) ≅ (ℤ/2)⁴、位数16。像は位数4なので全射ではない（16 > 4）。

(3) 「Hom の元のうち有理点から来ないもの」がまさに第2部・注意2（VIII.1.8）の非有効性の源。 第5部の言葉では：局所条件をすべて課して残る群が Selmer 群 S(2)(E/ℚ) で、 有理点由来の部分との差が Ш(E/ℚ)[2]。この曲線では実は S(2) がちょうど像と一致し Ш[2] = 0 となる（第5部の方法で計算可能）。つまり「余りの12元」は局所条件の段階で すでに排除される類である。
