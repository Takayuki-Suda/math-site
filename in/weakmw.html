大学院教材 ─ 楕円曲線の数論 2/5

弱 Mordell–Weil 定理
The Weak Mordell–Weil Theorem

E(K)/mE(K) は有限である──この一つの主張の中に、還元（局所理論）、Galois作用、 そして代数的整数論の2大有限性定理が全部詰まっています。 この第2部では AEC VIII §1 の証明を、行間を埋めながら一歩ずつ読みます。

TEXT: Silverman, AEC 2nd ed. ─ Ch. VIII §1（精読）

Ⅰ 全体地図 Ⅱ 弱Mordell–Weil Ⅲ Kummer対 Ⅳ 降下法 Ⅴ Selmer群とШ 終 その先へ 付 数論の武器庫
Section 01

主張と、その位置づけ

定理（弱Mordell–Weil） AEC VIII.1.1

K を数体、E/K を楕円曲線、m ≥ 2 を整数とする。このとき

E(K)/mE(K)　は有限群

まず、この定理が何を言っていないかをはっきりさせます。 「E(K)/mE(K) が有限」だけからは「E(K) が有限生成」は出ません。 たとえば ℝ/mℝ = 0 はどんな m でも自明群ですが、ℝ は有限生成ではありません。 同様に、p進体上の楕円曲線 E(ℚp) も E(ℚp)/mE(ℚp) は有限ですが E(ℚp) は有限生成になりませんAEC VIII §3 冒頭 / VII.6.3。 足りないのは「m で何回でも割れてしまう点」を排除する仕組みで、それを供給するのが高さ関数と降下法（第4部）です。

つまり Mordell–Weil 定理は「弱MW（今回）＋降下（第4部）」の2段ロケットで、 今回はその1段目を打ち上げます。

? なぜ商群 E(K)/mE(K) なのか

降下法（第4部の定理 VIII.3.1）が要求する入力がまさに「A/mA が有限」だからです。 降下法は「任意の点 P を、mで割った余り（＝剰余類の代表点）を引きながら縮めていく」議論なので、 余りの種類が有限でないと始まりません。弱MWは、その「余りの有限性」を保証します。

Section 02

証明の設計図

証明は4ステップです。先に全体を見てから、各ステップに潜ります。

図1：弱Mordell–Weil定理の証明構造。点線の箱が「外部から入ってくる深い定理」。 証明のオリジナリティはSTEP 1–2の「翻訳」にあり、有限性の供給源はSTEP 3の古典数論。

一言でまとめると：有理点の有限性の問題を、体拡大の有限性の問題に翻訳し（Kummer対）、 その体拡大が「行儀よい」ことを還元理論で示し（不分岐性）、 行儀よい拡大が有限個しかないことを古典数論で保証する（類数・単数）──これが全経路です。

Section 03 ─ Step 0

帰着補題 ─ E[m] ⊂ E(K) と仮定してよい

これから使う Kummer 対は「E[m] の点が K-有理」だと圧倒的に扱いやすくなります。 しかし一般の E/K では E[m] ⊄ E(K) です。そこで最初に、 体を有限次Galois拡大で取り替えても定理の成立・不成立は変わらないことを示します。

補題（帰着補題） AEC VIII.1.1.1

L/K を有限次Galois拡大とする。E(L)/mE(L) が有限ならば、E(K)/mE(K) も有限である。

証明（完全版）

包含 E(K) ⊂ E(L) から自然な写像 E(K)/mE(K) → E(L)/mE(L) が誘導されます。この核を Φ とおくと

Φ = ( E(K) ∩ mE(L) ) / mE(K)

です（「K有理点で、Lまで行けば m で割れるもの」たち）。 Φ が有限であることを示せば十分です。実際そのとき、完全列 0 → Φ → E(K)/mE(K) → E(L)/mE(L) により、E(K)/mE(K) は有限群 Φ と有限群 E(L)/mE(L) に挟まれて有限になります。

Φ の各元 P（mod mE(K)）に対し、[m]QP = P となる点 QP ∈ E(L) を1つ選び、写像

λP : GL/K → E[m],　λP(σ) = QPσ − QP

を定めます。まず行き先が本当に E[m] に入ること： [m](QPσ − QP) = ([m]QP)σ − [m]QP = Pσ − P = O。 最後の等号は P が K-有理で σ が K を固定することによります。

核心の主張：P ↦ λP は単射（Φ からの写像として）。 λP = λP′ と仮定すると、すべての σ ∈ GL/K で (QP − QP′)σ = QP − QP′。 つまり QP − QP′ は GL/K 不変、Galois理論より QP − QP′ ∈ E(K)。 m倍すると P − P′ = [m](QP − QP′) ∈ mE(K)、 すなわち P ≡ P′ (mod mE(K))。単射が示されました。

ところが GL/K も E[m] も有限集合なので、写像の集合 Map(GL/K, E[m]) は有限。 そこに Φ が単射で入るので Φ は有限です。

∎
? この証明の「変数」はどこにあるか

鍵は「P の m 等分点 QP のGalois軌道のずれ QPσ − QP が P の情報を（mod mE(K) で）覚えている」という一点です。 この λP は、第3部で学ぶ言葉では1-コサイクルそのものです。 AECも「λP is an example of a 1-cocycle; see (VIII §2)」と注記しています。 いま素手でやったこの議論は、第3部で inflation–restriction 完全列という一般論に整理されます。

さて、E[m] は位数 m² の有限集合で、各点の座標は K 内の代数的数ですAEC III.6.4。 そこで L = K(E[m])（全 m 分点の座標を添加した体）は有限次拡大であり、 GK/K が E[m] を集合として保つ（[m]P = O は Galois で不変な条件）ことから L/K は Galois 拡大です。帰着補題より、以後 E[m] ⊂ E(K) を仮定して定理を証明すれば十分です。 この仮定は §1 の残りすべてで有効です。

Section 04 ─ Step 1

Kummer 対 ─ 有理点を体拡大に翻訳する

定義（Kummer対） AEC VIII §1

E[m] ⊂ E(K) を仮定する。P ∈ E(K) と σ ∈ GK/K に対し、 [m]Q = P となる点 Q ∈ E(K) を任意に選び

κ(P, σ) = Qσ − Q

と定める。（Q の存在は [m] : E(K) → E(K) の全射性によるAEC III.6.2 + II.2.3。）

直観的には、κ(P, σ) は「P を m で割ろうとしたときに生じる Galois 的なねじれ」を測っています。 P が K の中で割り切れる（P ∈ mE(K)）なら Q を K-有理に取れるのでねじれは無し。 割り切れないなら、Q は真の拡大体に住み、σ が Q を動かした差分が E[m] の中に現れる──という構図です。

命題（Kummer対の基本性質） AEC VIII.1.2

(a) κ は well-defined。　(b) κ は双線形。　(c) 左核は mE(K)。　(d) 右核は GK/L。ここで

L = K([m]−1E(K)) = 「[m]Q ∈ E(K) となる全ての Q ∈ E(K) を添加した体」

したがって κ は完全対（perfect pairing）

E(K)/mE(K) × GL/K → E[m]

を誘導する。

証明（完全版・コメント付き）

(a) well-defined性。 まず κ(P,σ) ∈ E[m]：[m](Qσ − Q) = Pσ − P = O（P は K-有理）。 次に Q の選び方に依らないこと：別の選択は Q + T（T ∈ E[m]）の形しかない（差が m 倍で消えるから）。すると

(Q+T)σ − (Q+T) = Qσ − Q + (Tσ − T) = Qσ − Q

最後の等号で E[m] ⊂ E(K) を使いました：T が K-有理だから Tσ = T。 帰着補題で整えた仮定が、早速ここで効いています。

(b) 双線形性。 P について線形なのは定義から明らか（Q の選択を加法的に取ればよい）。 σ について：σ, τ ∈ GK/K に対し

κ(P, στ) = Qστ − Q = (Qσ − Q)τ + (Qτ − Q) = κ(P,σ)τ + κ(P,τ)

ここで κ(P,σ) ∈ E[m] ⊂ E(K) だから τ で不動、よって κ(P,στ) = κ(P,σ) + κ(P,τ)。 再び E[m] ⊂ E(K) が効いています。 この計算は「κ(P,·) は一般には準同型でなく1-コサイクル。E[m] が有理的なときに限り準同型になる」ことを示しており、 第3部（コホモロジー）での視点の伏線です。

(c) 左核 = mE(K)。 P = [m]Q, Q ∈ E(K) なら Qσ = Q なので κ(P,·) ≡ O。 逆にすべての σ で κ(P,σ) = O なら、Qσ = Q（∀σ ∈ GK/K）なので Q ∈ E(K)、 よって P = [m]Q ∈ mE(K)。

(d) 右核 = GK/L。 σ が L を固定すれば、各 Q（[m]Q ∈ E(K)）は L の点なので Qσ = Q、つまり κ(·,σ) ≡ O。 逆にすべての P で κ(P,σ) = O なら、[m]Q ∈ E(K) となる任意の Q に対し Qσ = Q。 L はそのような Q たちの座標で生成されるので σ は L を固定します。

最後に L/K が Galois であること：GK/K は集合 [m]−1E(K) を保つ （[m]Q ∈ E(K) ⇒ [m]Qσ = ([m]Q)σ ∈ E(K)）ので、L は共役で閉じています。 そして (d) から、σ ↦ κ(·,σ) が単射準同型

GL/K ↪ Hom( E(K), E[m] )

を与え、完全対の主張が従います。

∎
この時点で得られた「翻訳」

完全対の一般論より、E(K)/mE(K) は Hom(GL/K, E[m]) に単射で入ります。E[m] は有限なので：

E(K)/mE(K) が有限 ⟺ 拡大 L/K が有限

残る仕事は「L = K([m]−1E(K)) は K の有限次拡大」を示すことだけになりました。 有理点の問題が、体の問題になった──これがSTEP 1の成果です。

古典Kummer理論との対応（アナロジーの辞書）

「Kummer対」という名前の由来です。μm ⊂ K を含む体の古典的な Kummer 理論と、 いま作った理論は完全に平行していますAEC VIII.1.2.1。

古典Kummer理論（乗法群）楕円曲線版（今回）
K× と m乗写像 x ↦ xmE(K) と m倍写像 [m]
μm（1のm乗根）⊂ K を仮定E[m]（m分点）⊂ E(K) を仮定
K×/(K×)mE(K)/mE(K)
m√a を添加した体 K′[m]−1E(K) を添加した体 L
(a,σ) ↦ (m√a)σ / m√a ∈ μm(P,σ) ↦ Qσ − Q ∈ E[m]

なお、E[m] ⊂ E(K) のとき Weil 対の系AEC III.8.1.1により μm ⊂ K が自動で従うので、 楕円曲線版の仮定は古典版の仮定を含んでいます。この辞書は第3部でコホモロジーの言葉に昇格します。

Section 05 ─ Step 2

拡大 L/K の性質 ─ 分岐は S の外で禁止される

まず、悪さをしうる素点の集合を固定します。

S = { v : E が v で悪い還元 } ∪ { v : v(m) ≠ 0 } ∪ MK∞
MK∞ = 無限素点（アルキメデス的絶対値）の集合

S が有限集合であることが大前提です：Weierstrass方程式を1本固定すると、 有限個を除くすべての v で係数が v-整かつ v(Δ) = 0 となり、そこでは良い還元を持ちますAEC VIII.1.3。 m を割る素点も有限個、無限素点も有限個です。

Ch.VIIから持ってきた武器を、使う形で再掲します。

命題（m分点の還元は単射） AEC VIII.1.4 = VII.3.1(b)

v を v(m) = 0 なる有限素点とし、E は v で良い還元を持つとする。このとき還元写像

E(Kv)[m] → Ẽv(kv)

は単射である。（核となる E₁ は形式群で制御され、そこには m と素な捩れがない〔AEC IV §6, VII.2.2〕。）

命題（Lの性質） AEC VIII.1.5

L = K([m]−1E(K)) とする。

(a) L/K はアーベル拡大で、Galois群の指数（exponent）は m を割る。

(b) L/K は S の外で不分岐。すなわち v ∉ S ならば L/K は v で不分岐。

証明（完全版・コメント付き）

(a) STEP 1 の単射 GL/K ↪ Hom(E(K), E[m]) から直ちに従います。 右辺はアーベル群で、すべての元の m 倍が消える（値域が E[m] だから）ので、 GL/K もアーベルかつ指数 m です。

(b) ここが急所です。v ∉ S を取り、[m]Q ∈ E(K) となる Q を1つ選び、K′ = K(Q) とおきます。 L はこのような K′ たちの合成体なので、各 K′/K が v で不分岐であることを示せば十分です （不分岐拡大の合成は不分岐）。

v′ を K′ の v の上の素点とし、惰性群 Iv′/v ⊂ GK/K の元 σ を任意に取ります。 「K′/K が v′ で不分岐」とは「惰性群が K′ に自明に作用する」ことなので、 σ が Q を動かさないことを示せばよいのです。

v ∉ S より E は v で良い還元を持ち（同じWeierstrass方程式が v′ でも最小・非特異）、還元写像 E(K′v′) → Ẽ(k′v′) があります。惰性群の定義そのものにより、σ は剰余体 k′v′ に自明に作用します。したがって

(Qσ − Q)˜ = Q̃σ − Q̃ = Q̃ − Q̃ = Õ　（還元先ではズレが見えない）

一方、[m](Qσ − Q) = ([m]Q)σ − [m]Q = O（[m]Q は K-有理）なので Qσ − Q ∈ E[m]。 つまり Qσ − Q は「m分点であり、かつ還元の核に入る点」です。 v(m) = 0 だから VIII.1.4 が適用でき、そのような点は O しかありません：

Qσ − Q = O,　すなわち Qσ = Q

よって惰性群は Q を固定し、K′ = K(Q) は v で不分岐です。 これがすべての v ∉ S と、v の上のすべての v′ について成り立つので (b) が従います。

∎
? この議論の何がすごいのか

「L/K の分岐」は純粋に大域的・体論的な情報です。それを、 「還元した曲線の上では惰性群は何も動かせない」という幾何的な事実が禁止する── 局所（各素点での還元）が大域（体拡大の形）を縛る、数論幾何の典型的な構図です。 なお (b) の証明は E[m] ⊂ E(K) の仮定を使っていません（AECもこの点を注意しています）。 この事実は第5部で Selmer 群の有限性を示すとき、そのまま再利用されます。

Section 06 ─ Step 3

体論の有限性 ─ 数論の心臓部

命題 AEC VIII.1.6

K を数体、S ⊂ MK を MK∞ を含む有限集合、m ≥ 2 とする。 「指数 m のアーベル拡大で S の外不分岐なもののうち最大のもの」を L/K とすると、L/K は有限次拡大である。

証明は5段階です。各段階でどの古典定理が働くかに注目してください。

証明（完全版・コメント付き）

(i) μm ⊂ K と仮定してよい。 K′ = K(μm)、S′ を S の上の素点全体とします。命題が K′ で正しければ、 LK′/K′ は指数m・アーベル・S′外不分岐なので有限次、よって L/K も有限次です。 以後 μm ⊂ K とします。

(ii) S を大きくしてよい。S を大きくすると L は大きくなるだけなので、大きい S で示せば十分。 そこでイデアル類群の有限性を使い、類群の代表イデアル a₁, …, ah を割る素点を S に加えて、 S-整数環

RS = { a ∈ K : v(a) ≥ 0 （∀v ∉ S） }

が単項イデアル整域（PID）になるようにします。さらに v(m) = 0（∀v ∉ S）となるよう m を割る素点も加えます。

(iii) Kummer理論。μm を含む標数0の体では、 最大指数mアーベル拡大は「全元のm乗根の添加」で得られます（体論の教科書、または AEC演習8.4）。 よって L は K(m√a : a ∈ K×) の中の「S外不分岐な最大部分体」です。 どの m√a が不分岐拡大を与えるか判定します。 v ∉ S では v(m) = 0 で、多項式 Xm − a の判別式は ±mmam−1 なので：

Kv(m√a)/Kv が不分岐 ⟺ ordv(a) ≡ 0 (mod m)

したがって、L を生成するのは次の集合の元のm乗根だけです：

TS = { a ∈ K×/(K×)m : ordv(a) ≡ 0 (mod m) （∀v ∉ S） }
L = K(m√a : a ∈ TS)

あとは TS が有限集合であることを示せば、Kummer理論により [L : K] も有限です。

(iv) S-単数が TS に全射する。 自然な写像 RS× → TS を考えます。a ∈ K× が TS の元を代表するとします。 条件 ordv(a) ≡ 0 (mod m)（∀v∉S）は「イデアル aRS が RS のイデアルの m 乗」を意味します （RS の素イデアル ⟷ v ∉ S）。RS は PID なので aRS = bmRS となる b ∈ K× があり、 u = a/bm とおくと u ∈ RS× かつ、K×/(K×)m の中で a と u は同じ類です。 よって全射

RS×/(RS×)m ↠ TS

を得ます（実は同型ですが、全射で十分）。

(v) DirichletのS-単数定理。RS× は有限生成 （具体的には μ(K) × ℤ#S−1）なので、RS×/(RS×)m は有限、 よって TS は有限。以上で証明が完了します。

∎
定量的な補足（効率の感覚）

(v) の構造定理から |RS×/(RS×)m| = m#S （μm ⊂ K のとき）なので、拡大後の S に対して [L : K] ≤ m#S という明示的な上界が出ます。 つまり弱MW定理は「E(K)/mE(K) の位数は m2·(何か計算可能な数) 以下」という形の 明示的な上界まで与えてくれます。

? なぜ「類数」と「単数」の両方が要るのか

TS の元は「付値が全部 m の倍数」という条件しか課されていません。 これを「m乗×単数」の形に直すとき、イデアルのレベルで m 乗根が取れること（PID性＝類群を殺した状態）と、 残った単数部分が有限型であること（S-単数定理）の両方が必要になります。 類数の有限性が「イデアルと元のズレ」を制御し、単数定理が「元の可逆部分」を制御する── 代数的整数論の2大有限性定理が、それぞれ別の役割で正確に一度ずつ使われる。美しい分業です。

Section 07

証明の完成と、2つの注目すべき注意

弱Mordell–Weil定理の証明（組み立て）

帰着補題（STEP 0）により E[m] ⊂ E(K) と仮定する。L = K([m]−1E(K)) とおく。 完全対（STEP 1）より、E(K)/mE(K) が有限であることと GL/K が有限であることは同値。 STEP 2 より L/K は指数mのアーベル拡大で有限集合 S の外不分岐。 STEP 3 より、そのような拡大は有限次。よって E(K)/mE(K) は有限。

∎（AEC VIII.1.1）

注意1：Minkowski 経由の別証明 AEC VIII.1.7

STEP 3 の代わりに、Minkowski–Hermite の定理 「次数が有界で S の外不分岐な K の拡大は有限個」を使う道もあります。 実際、Q ∈ [m]−1E(K) の Galois 共役はすべて Q + T（T ∈ E[m]）の形なので [K(Q) : K] ≤ m² と次数が一様に抑えられ、 K(Q) の候補が有限個 ⇒ 合成体 L は有限次、となります。 この見方は「なぜ有限になるのか」の別の直観（判別式の有界性）を与えてくれます。

注意2：どこが「計算できない」のか AEC VIII.1.8

実は GL/K は（上の証明を明示的にたどれば）完全に計算可能です。 すると単射 E(K)/mE(K) ↪ Hom(GL/K, E[m]) の右辺の元もすべて書き下せます。 計算できないのはただ一点、「右辺のどの元が、実際に有理点から来るか」の判定です。 Mordell–Weil定理の非有効性はすべてここに集中しており、 この一点を攻めるために発明されたのが Selmer 群です（第5部）。 この注意は必ず頭に置いてから第5部に進んでください。

Section 08

実例で全部を確かめる ─ E : y² = x³ − x, m = 2

抽象論を1つの曲線で最後まで実行します。K = ℚ、m = 2。 この曲線は Δ = 64、E[2] = {O, (0,0), (1,0), (−1,0)} がすべて ℚ-有理なので、 帰着補題なしでいきなり理論が回ります。さらに E(ℚ) はこの4点で全部です （階数0。これは第5部の方法で証明できるので、ここでは認めます）。

S = {2, ∞}　（Δ = 64 なので悪い還元は 2 のみ。m = 2 も 2 のみを割る）

L = ℚ([2]⁻¹E(ℚ)) を実際に計算する

各 P ∈ E(ℚ) に対し [2]Q = P の解 Q を求めます。倍加公式AEC III.2.3(d)は A = −1, B = 0 なので：

x([2]Q) = (x² + 1)² / ( 4(x³ − x) )　（x = x(Q)）

たとえば P = (1, 0) の場合。x([2]Q) = 1 を整理すると x⁴ − 4x³ + 2x² + 4x + 1 = 0。これは見事に平方の形に畳めます：

x⁴ − 4x³ + 2x² + 4x + 1 = (x² − 2x − 1)²　⇒　x = 1 ± √2
y² = x³ − x = 2(1+√2)²　⇒　y = ±√2(1+√2)　（x = 1+√2 のとき）

つまり (1,0) の半分の点は ℚ(√2) に住んでいます。他の点も同様に計算すると：

P ∈ E(ℚ)Q（[2]Q = P の解の一つ）K(Q)
OQ ∈ E[2] ⊂ E(ℚ)ℚ
(0, 0)(i, 1−i) などℚ(i)
(1, 0)(1+√2, √2(1+√2)) などℚ(√2)
(−1, 0)(−1+√2, i√2(√2−1)) などℚ(i, √2)
L = ℚ(i, √2),　GL/ℚ ≅ (ℤ/2ℤ)²

理論の予言と照合する

  • 指数2のアーベル拡大か？ Gal ≅ (ℤ/2)²。✓
  • S = {2,∞} の外で不分岐か？ ℚ(i,√2) = ℚ(ζ₈)（1の8乗根の体）で、その判別式は 2 のべき。 分岐する有限素点は 2 のみ。✓ 理論（VIII.1.5b）の通りです。
  • 完全対になっているか？ |E(ℚ)/2E(ℚ)| = 4、|GL/ℚ| = 4、E[2] ≅ (ℤ/2)²。 4×4 の対が (ℤ/2)² に値を取り、両側の核が消えている──サイズも整合します。✓

抽象的な L = K([m]−1E(K)) が、実際にはこんなに小さく具体的な体（円分体 ℚ(ζ₈)！）になる。 この感覚を持っておくと、第5部で K(S,2) = {±1, ±2} のような有限集合が出てきたときに驚かずに済みます。

Exercises

練習問題

★=基礎（定義と証明の確認）、★★=標準（手を動かす）、★★★=発展（自力で議論を組む）。 すべての問題に解答または方針が付いています。まず自力で15分、が目安です。

問 1★ 基礎

Kummer対の well-defined 性の証明（VIII.1.2a）では、仮定 E[m] ⊂ E(K) が1箇所だけ本質的に使われた。 その箇所を特定せよ。また、この仮定を外すと κ(P,σ) はどんな対象として生き残るか （ヒント：値は Q の選択に依存してどう変わるか）。

解答を開く

使われたのは「別の選択 Q + T（T ∈ E[m]）に対し Tσ = T」の部分。 E[m] ⊂ E(K) だから σ が T を固定できた。 仮定を外すと、Q を Q + T に取り替えたとき κ の値は (Q+T)σ − (Q+T) = (Qσ − Q) + (Tσ − T) となり、 Tσ − T の分だけずれる。σ ↦ Tσ − T はコバウンダリ（自明な1-コサイクル）なので、 κ(P,·) は「写像」としては well-defined でなくなるが、 1-コサイクルのコホモロジー類としては well-defined のまま生き残る。 これが第3部で δ : E(K)/mE(K) → H¹(G, E[m]) が一般の状況で定義できる理由である。

問 2★ 基礎

帰着補題の証明中、λP = λP′ ⇒ P ≡ P′ (mod mE(K)) の議論を、 本文を見ずに自分で再構成せよ。とくに「Galois不変な K-点は K-点」という事実を どこで使うか明示すること。

解答を開く

λP(σ) = λP′(σ) ⟺ QPσ − QP = QP′σ − QP′ ⟺ (QP − QP′)σ = QP − QP′。 これが全ての σ ∈ GL/K で成立。QP, QP′ ∈ E(L) なので差も E(L) の点であり、 GL/K 不変な L-点は（Galois理論により座標が K に落ちるので）K-点。 よって R := QP − QP′ ∈ E(K)。 [m]R = P − P′ なので P − P′ ∈ mE(K)。

問 3★★ 標準

不分岐性の証明（VIII.1.5b）では v(m) = 0（v が m を割らない）という条件が必要だった。 この条件が外れる（p = char kv が m を割る）と、証明のどのステップが崩れるか。 形式群（AEC IV §6）の言葉で説明せよ。

解答を開く

崩れるのは「Qσ − Q ∈ E[m] ∩ (還元の核) ⇒ Qσ − Q = O」のステップ（VIII.1.4 の適用）。 還元の核 E₁ は形式群 Ê(M) と同型で、Ê(M) には p と素な位数の捩れ元は存在しないが、 p ベキ位数の捩れ元は存在しうる（例：ℚp の分岐拡大上）。 p | m だと E[m] が p-部分を持つため、「m分点かつ還元の核」という点が O 以外にありえて、 単射性が保証されなくなる。実際、m = p のとき K([p]⁻¹E(K))/K は良い還元の素点 p の上でも分岐しうる。 S に「m を割る素点」を入れておくのはこのためである。

問 4★★ 標準

E : y² = x³ − x, P = (−1, 0) とする。[2]Q = P を満たす Q の x座標が満たす4次方程式を導き、 それが (x² + 2x − 1)² であることを確かめ、Q の座標と体 ℚ(Q) を決定せよ （本文の表の最終行を自力で埋める問題）。

解答を開く

倍加公式より (x²+1)² / (4(x³−x)) = −1、整理して x⁴ + 4x³ + 2x² − 4x + 1 = 0。 展開比較で (x² + 2x − 1)² = x⁴ + 4x³ + 2x² − 4x + 1 を確認（(x²+2x−1)² の展開：x⁴+4x³+4x²−2x²−4x+1）。 よって x = −1 ± √2。x = −1+√2 のとき y² = x³ − x = x(x−1)(x+1) = (−1+√2)(−2+√2)(√2) = −2(√2−1)² < 0（実数では負）なので y = ± i√2(√2−1)。したがって Q = (−1+√2, i√2(√2−1)) で、ℚ(Q) = ℚ(i, √2)（4次拡大）。 この P だけは半分の点が ℚ(i) にも ℚ(√2) にも入らず、合成体が必要になる。

問 5★★ 標準

VIII.1.6 の証明では全射 RS×/(RS×)m ↠ TS を作ったが、 AECは「実は同型」と注記している。単射性を証明せよ。 すなわち u ∈ RS× が K× の中で m乗（u = bm, b ∈ K×）なら、 b ∈ RS× であることを示せ。

解答を開く

任意の v ∉ S に対し m·ordv(b) = ordv(bm) = ordv(u) = 0 （u は S-単数なので）。よって ordv(b) = 0 が全ての v ∉ S で成り立ち、b ∈ RS×。 つまり u が (K×)m に入るなら既に (RS×)m に入っており、 核は自明。全射と合わせて RS×/(RS×)m ≅ TS。

問 6★★★ 発展

（Minkowski経由の別証明AEC VIII.1.7を完成させる） E[m] ⊂ E(K) を仮定する。Q ∈ [m]−1E(K) に対し、拡大次数の評価 [K(Q) : K] ≤ m² を証明せよ。さらに、Hermite–Minkowskiの定理 「次数有界・S外不分岐な拡大は有限個」を認めて、L = K([m]−1E(K)) が有限次であることを導け。

解答の方針を開く

次数評価：σ ∈ GK/K に対し [m](Qσ − Q) = ([m]Q)σ − [m]Q = O なので Qσ ∈ { Q + T : T ∈ E[m] }。 つまり Q の Galois 共役は高々 #E[m] = m² 個しかなく、Q の（座標の）最小多項式の次数は m² 以下。 よって [K(Q) : K] ≤ m²。

有限性：各 K(Q) は VIII.1.5b と同じ議論で S の外不分岐（この議論は E[m] ⊂ E(K) を使わない）。 次数 ≤ m² かつ S外不分岐な拡大は Hermite–Minkowski により有限個しかないので、 体の候補 K(Q₁), …, K(Qt) は有限個。L はそれらの合成体なので [L : K] ≤ Π [K(Qi) : K] < ∞。

比較：本文の証明（Kummer理論）は「アーベル・指数m」という構造を使って有限性を出したのに対し、 こちらは「次数の一様有界性」だけで押し切る。どちらも「不分岐性」（STEP 2）は共通に必要である点に注意。
