極限とε-δ論法

高校生〜大学1年 ・ 解析学入門 ／ 「近づく」を厳密に定義する

0ε-δ定義とは何か
limx→a f(x) = L の厳密な定義は以下のとおりです。
∀ε > 0,  ∃δ > 0  s.t.  0 < |x − a| < δ  ⟹  |f(x) − L| < ε
「ε（出力誤差の上限）を相手がどう小さく決めても、
それに応じた δ（入力の許容幅）を自分が見つけられる」
↓ スライダーで ε を変えて、δ が自動で決まる様子を確かめよう（f(x)=2x, a=1, L=2）。
ε（相手が決める）
0.70
δ = ε/2（自動計算）
0.35
1ε-δゲーム ── f(x)=x²、a=2、L=4
相手（数学の神）が ε を設定します。
あなたは δ を選んで、定義の条件を満たしてください。
「0 < |x−2| < δ のすべての x で |x²−4| < ε が成り立つ」
ε（相手が設定）
0.80
δ（あなたが選ぶ）
0.50
判定中…
2不連続点では ε-δ を満たす δ が存在しない
符号関数 sgn(x)（x>0 なら 1、x<0 なら −1）は x=0 で不連続です。
ε = 0.5 と固定したとき、どんな δ を選んでも条件を満たせません。
δ（試してみる）
0.75
常に不成立
3ε-δ 証明の組み立て方
limx→1 2x = 2 の証明を、ε-δ の手順で組み立てます。
1
ε を受け取る：任意の ε > 0 が与えられる。
2
δ を構成する：δ = ε/2 と定める。
3
条件を検証する：0 < |x−1| < δ と仮定する。
|f(x) − 2| = |2x − 2| = 2|x − 1| < 2δ = 2·(ε/2) = ε  ✓
4
結論：∀ε>0 に対して δ=ε/2 を取れば定義の条件が成り立つ。
ゆえに limx→1 2x = 2  □

ε-δ論法の核心 — 4つのポイント

∀ε > 0（先攻：相手）
相手がどんなに小さな ε を出してきても対応できなければならない。「任意」が鍵。
∃δ > 0（後攻：自分）
ε に依存した δ を「構成」できれば勝ち。存在を示すには具体的に作ればよい。
0 < |x−a| < δ
x=a 自身（穴あき近傍）を除く。f(a) の値は極限値に無関係。
不連続 → 極限なし
ε を小さくしたとき対応する δ が存在しない ⟺ 極限が存在しない（または ≠L）。
