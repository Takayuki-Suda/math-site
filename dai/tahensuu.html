多変数解析・微分形式

大学生 ・ 解析学 ／ 勾配・発散・回転・ストークスの定理

0勾配（gradient）――最も急な登り坂の方向
スカラー場 f(x,y) に対して grad f = (∂f/∂x, ∂f/∂y) は「最も急に増加する方向」を指します。
スライダーで観察点を動かして確かめよう。
観察点 x
1.0

0.5
1発散（divergence）――源・吸い込み
ベクトル場 F=(P,Q) に対して div F = ∂P/∂x + ∂Q/∂y。
流体の「湧き出し量」を表します。
2回転（curl）――渦の強さ
2次元では curl F = ∂Q/∂x − ∂P/∂y。
渦の強さと向きを表します。スライダーで渦の強さを変えよう。
渦の強さ
1.0
3ストークスの定理
∮∂S F·ds = ∬S (curl F)·dA
「境界での積分」=「内部での積分」。これがガウスの定理・グリーンの定理の一般化です。
ストークスの定理の直感：
境界（曲線）に沿った流れの合計 = 内部の渦（curl）の総量。
「局所の渦」を全部足すと「大域の周回」になる。
→ グリーンの定理（平面）・発散定理（体積）はこれの特殊ケース。

多変数解析の核心

grad f = ∇f
スカラー場の「最急上昇方向」。等高線と直交する。
div F = ∇·F
ベクトル場の「湧き出し量」。正なら源、負なら吸い込み。
curl F = ∇×F
ベクトル場の「渦の強さと向き」。保存場では curl=0。
ストークスの定理
∮∂S F·ds = ∬S (∇×F)·dA。境界と内部をつなぐ橋。
