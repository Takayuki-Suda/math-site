Tate–Shafarevich 群
The Selmer and Shafarevich–Tate Groups

Mordell–Weil 定理はどこで「計算可能」で、どこで「計算不能」になるのか。 その境界線を正確に描くのが Selmer 群と Ш 群です。 AEC X §4 を精読し、実際に1本の曲線の Mordell–Weil 群を計算しきります。 最後に Ch. IX と Ch. X の残りを展望して、シリーズを締めくくります。

Section 01

問題の再設定 ─ 失われた計算可能性を取り戻す

シリーズのここまでで確定した状況を整理します。

  • E(K) は有限生成（第4部）。しかも E(K)/mE(K) の生成元さえ分かれば、 E(K) の生成元は降下法で有効に計算できるAEC VIII.3.2 / 演習8.18。
  • E(K)/mE(K) は有限で、計算可能な有限群 H¹(GK/K, E[m]; S) （S外不分岐類）に単射で入る（第2・3部）。
  • ただし「H¹ のどの元が有理点由来か」の判定法がないAEC VIII.1.8。 ここが唯一の非有効ポイント。

Chapter X の発想はこうです：「有理点由来か」は判定できないが、 「局所的に有理点由来でありうるか」は判定できる。 各完備化 Kv は Hensel の補題のおかげで「計算しやすい世界」だからです。 そこで H¹ を局所条件で切った群──Selmer群──を間に挟み、 切り残った誤差を Ш と名付けて正面から研究する。これが X §4 の設計です。

Section 02 ─ 準備（要約）

X §1–§3 の道具立て ─ 等質空間という辞書

X §1：完全 2-降下（すでに半分知っている）

第3部 §7 で見た「E[2] ⊂ E(K) のとき E(K)/2E(K) ↪ (K×/(K×)²)²、 P ↦ (x−e₁, x−e₂)」の像を特定する話ですAEC X.1.1 / X.1.4。 鍵は、像が有限群

K(S, m) = { b ∈ K×/(K×)m : ordv(b) ≡ 0 (mod m)（∀v ∉ S） }
第2部STEP 3の TS と同じもの。S = {悪い還元} ∪ {v | 2} ∪ MK∞

の2つ組に入ること、そして (b₁,b₂) ∈ K(S,2)² が像に入るか否かが補助曲線の有理点の存在

b₁z₁² − b₂z₂² = e₂ − e₁,　b₁z₁² − b₁b₂z₃² = e₃ − e₁

に言い換えられることですAEC X.1.4。 例として AEC は y² = x(x−2)(x−10) を完全に計算し、階数0を示していますAEC X.1.5。 「E(K)/2E(K) の計算 ＝ 有限個の補助曲線たちの有理点探し」という形が、この章全体の原型です。

X §2–§3：twist・等質空間・Weil–Châtelet 群

第3部で「H¹(G, E(K)) ≠ 0 が楕円曲線の理論を難しくしている」と言いました。 X §3 はこの群に幾何的な意味を与えます。

定義（等質空間） AEC X §3

E/K の（主）等質空間（principal homogeneous space, torsor）とは、 種数1の滑らかな曲線 C/K と、K上定義された単純推移的な作用 C × E → C の組。 自明な例は C = E 自身。2つの等質空間は K-同型で同一視し、同値類全体を Weil–Châtelet 群 WC(E/K) と呼ぶ。

使う事実は次の3つだけです（証明は読み流してよい）：

  • {C/K} が自明 ⟺ C(K) ≠ ∅AEC X.3.3。 「K-有理点を持つ torsor は E と同一視できる」。
  • WC(E/K) ≅ H¹(GK/K, E(K))AEC X.3.6。 対応は、K-同型 θ : C → E を選び σ ↦ θσ(p) − θ(p) 型のコサイクルを取るもの。
  • C の Jacobian（Pic⁰(C)）は E に戻るAEC X.3.8。 「E の torsor ＝ Jacobian が E である種数1曲線」という辞書。
コホモロジーの言葉幾何の言葉
ξ ∈ H¹(G, E)種数1曲線 C/K（Jacobian が E）
ξ = 0C(K) ≠ ∅（有理点がある）
resv(ξ) = 0 ∈ H¹(Gv, E)C(Kv) ≠ ∅（v進点がある）
⏩ ここで止まらない

X §2（twist の一般論）と X §3 の証明群は、初読では上の辞書（3項目＋表）だけ持って通過してよい部分です。 辞書さえ手にあれば §4 の議論はすべて追えます。等質空間の理論の詳細が必要になったら AEC X §3 に戻ってください。

Section 03

基本図式と2つの群の定義 AEC X §4

φ : E → E′ を K 上定義された同種写像とします（E = E′、φ = [m] が典型例。 一般化しておくのは「2-同種による降下」（§5）で本当に使うからです）。 短完全列 0 → E[φ] → E → E′ → 0 から、第3部と同じ手順で大域の基本完全列：

0 → E′(K)/φ(E(K)) →δ H¹(GK/K, E[φ]) → WC(E/K)[φ] → 0　(∗)

各素点 v で同じことをすると（Gv ⊂ GK/K は分解群）、局所版：

0 → E′(Kv)/φ(E(Kv)) →δ H¹(Gv, E[φ]) → WC(E/Kv)[φ] → 0　(∗v)

そして制限写像で両者を束ねた可換図式が、この節の全議論の舞台です：

図1：基本可換図式 (∗∗)〔AEC X §4〕。上段が大域 (∗)、下段が局所 (∗v) の積。 目標＝上段左（有理点）の像の決定。右下（局所WC）は Hensel の補題により計算可能。
定義（Selmer群・Tate–Shafarevich群） AEC X §4
S(φ)(E/K) = ker{ H¹(GK/K, E[φ]) → ∏v WC(E/Kv) }　（φ-Selmer群）
Ш(E/K) = ker{ WC(E/K) → ∏v WC(E/Kv) }　（Tate–Shafarevich群）

いずれも v の K への延長の選び方に依らないAEC X.4.1.1。

幾何の辞書（§2）で読み替えると、定義の意味が鮮明になります：

  • S(φ) ＝「対応する等質空間がすべての完備化で点を持つようなコホモロジー類」。 つまり「局所的には有理点由来と区別がつかない類」の集まり。
  • Ш ＝「どこでも局所的に自明なのに、大域的には自明でないtorsorたち」AEC X.4.1.2。 つまりHasse原理（局所大域原理）の破れを測る群。種数0の曲線（円錐曲線）では Hasse–Minkowski の定理によりこの群に相当するものは消えますが、種数1では消えません。
Section 04

基本定理 ─ 完全列と有限性

定理 AEC X.4.2

φ : E/K → E′/K を同種写像とすると：

(a)　0 → E′(K)/φ(E(K)) → S(φ)(E/K) → Ш(E/K)[φ] → 0　は完全
(b)　S(φ)(E/K) は有限群
(a) の証明（図式追跡を全部書く）

像が Selmer に入ること：P ∈ E′(K) に対し δ(P) は大域列 (∗) の完全性より WC(E/K) で 0 に行く。図式の可換性より、各 v での像（∈ WC(E/Kv)）も 0。よって δ(P) ∈ S(φ)。

S(φ) → Ш[φ] が定義できること：ξ ∈ S(φ) の WC(E/K)[φ] での像を c とすると、 可換性より c の各局所像は ξ の局所像と一致し、それは 0。よって c ∈ Ш[φ]。

全射性：c ∈ Ш(E/K)[φ] を任意に取る。大域列 (∗) の完全性（右端が全射）より c はある ξ ∈ H¹(G, E[φ]) の像。ξ の局所像は c の局所像 = 0 に行くので ξ ∈ S(φ)。

S(φ) での完全性：ξ ∈ S(φ) が Ш[φ] で 0 に行く ⟺ ξ の WC(E/K) での像が 0 ⟺（大域列の完全性）ξ ∈ im δ。

∎
(b) の証明（弱MWの議論の再演）

ξ ∈ S(φ) とし、v を「deg φ = m を割らず、E が良い還元を持つ」有限素点とします。 ξ が v で不分岐であることを示します。

ξ ∈ S(φ) より ξ は WC(E/Kv) で自明。局所列 (∗v) の完全性から、 ある P ∈ E(Kv) が存在して ξσ = Pσ − P（∀σ ∈ Gv）、 しかも Pσ − P ∈ E[φ] ⊂ E[m]。σ を惰性群 Iv の元とすると、 還元先では惰性は動かないので (Pσ − P)˜ = Õ。 つまり Pσ − P は「m分点かつ還元の核」の点であり、 VIII.1.4（= VII.3.1、第2部STEP 2の武器）よりそれは O。 よって ξσ = 0（∀σ ∈ Iv）、ξ は v で不分岐です。

したがって S(φ) ⊂ H¹(G, E[φ]; S)（S = {悪い還元} ∪ {v | deg φ} ∪ MK∞、 有限集合AEC X.4.4。E と E′ の悪い還元素点が一致すること〔AEC VII.7.2〕もここで効く）。 あとは次の補題で終わりです。

∎（補題を除き）
補題（S外不分岐コホモロジーの有限性） AEC X.4.3

M を有限 GK/K-加群、S を有限な素点集合とすると、 H¹(GK/K, M; S) = { ξ : S外で不分岐 } は有限。

証明

M は有限なので、M の全元を固定する開部分群がある。inflation–restriction（第3部）により K を有限次拡大に取り替えてよく、作用は自明としてよい。 すると H¹(G, M; S) = Hom(G, M; S)。M の指数を m とし、 L/K を「指数mの最大アーベル拡大で S外不分岐なもの」とすると、 S外不分岐な準同型 G → M はすべて GL/K を経由する。 VIII.1.6（第2部STEP 3！）より L/K は有限次なので、Hom(GL/K, M) は有限。

∎
? 「弱MWの証明」と「Selmer群の有限性」は同じ定理である

φ = [m] の場合、(a) の完全列の左端が E(K)/mE(K) なので、(b) の有限性から弱MW定理が従います。 逆に (b) の証明は、第2部の STEP 2（不分岐性）と STEP 3（体論の有限性）をそのまま再演したものです。 つまり弱Mordell–Weil定理の証明の「本体」は Selmer 群の有限性であり、 第2部で学んだことは一切無駄になっていない。シリーズ全体がここで一つに繋がります。

Selmer群は計算可能 AEC X.4.5

H¹(G, E[φ]; S) は（第2部STEP 3を明示的にたどれば）計算可能な有限群。 その各元 ξ について「ξ ∈ S(φ) か？」は、対応する等質空間 Cξ が 各 v ∈ S で Kv-点を持つかの判定であり、Henselの補題により有限回の合同式チェックに帰着する （v ∉ S では不分岐性から自動的に局所点がある）。 よって S(φ)(E/K) は原理的に必ず計算できる。 計算できないのは Ш[φ]──「局所的に見えない大域的障害」だけである。

Section 05

2-同種による降下 ─ 実戦用の道具 AEC X.4.8–4.9

K-有理な2分点 T = (0,0) を持つ曲線なら、[2] を「次数2の同種写像2つの合成」に分解して 降下計算を軽くできます。E(K)[2] ≠ 0 なら常に使える、実戦上最も重要な形ですAEC X.4.9.2。

命題（2-同種による降下） AEC X.4.9
E : y² = x³ + ax² + bx,　E′ : Y² = X³ − 2aX² + (a² − 4b)X
φ : E → E′,　φ(x,y) = ( y²/x², y(b − x²)/x² ),　ker φ = { O, (0,0) },　deg φ = 2
S = MK∞ ∪ { v : v(2) ≠ 0 または v(b) ≠ 0 または v(a²−4b) ≠ 0 }

d ∈ K× に対し等質空間 Cd を

Cd : dw² = d² − 2adz² + (a² − 4b)z⁴

で定めると、完全列

0 → E′(K)/φ(E(K)) →δ K(S,2) → WC(E/K)[φ]
δ(O) = 1,　δ((0,0)) = a² − 4b,　δ((X,Y)) = X（X ≠ 0）／　d ↦ {Cd/K}

があり、Selmer群は

S(φ)(E/K) ≅ { d ∈ K(S,2) : Cd(Kv) ≠ ∅（∀v ∈ S） }

さらに ψ : Cd → E′, ψ(z,w) = ( d/z², −dw/z³ ) は、 P ∈ Cd(K) が見つかったとき δ(ψ(P)) ≡ d を満たす（点の回収装置AEC X.4.6）。

E′ は E と同じ形なので、双対同種写像 φ̂ : E′ → E にも同じ命題が使えます。 2つの降下の結果は、次の完全列で E(K)/2E(K) に合成されますAEC X.4.7：

0 → E′(K)[φ̂] / φ(E(K)[2]) → E′(K)/φ(E(K)) →φ̂ E(K)/2E(K) → E(K)/φ̂(E′(K)) → 0
（φ̂∘φ = [2] を使った初等的な図式追跡で得られる）
Section 06 ─ 完全実行例

実戦：E : y² = x³ − 6x² + 17x の Mordell–Weil 群 AEC X.4.10

a = −6, b = 17。判別式は Δ = −2⁹·17² なので S = {∞, 2, 17}、 ℚ(S,2) = {±1, ±2, ±17, ±34}（8元）。2-同種で移る相手は

E′ : Y² = X³ + 12X² − 32X,　Cd : dw² = d² + 12dz² − 32z⁴

φ 側の Selmer 群を決める

  • d = −2 ∈ S(φ)：(0,0) ∈ E′(ℚ) の像が δ(0,0) = a²−4b = −32 ≡ −2。 有理点由来なので自動的に Selmer に入る。
  • d = 2 ∈ S(φ)（しかも有理点由来）：C₂ : 2w² = 4 + 24z² − 32z⁴ を z = Z/2 で整理すると w² = 2 + 3Z² − Z⁴。目視で有理点 (Z, w) = (1, 2)、 すなわち (z,w) = (1/2, 2) ∈ C₂(ℚ)。回収装置 ψ で ψ(1/2, 2) = (8, −32) ∈ E′(ℚ)、検算：δ(8,−32) = 8 ≡ 2。✓
  • d = 17 ∉ S(φ)（17進で死ぬ）：C₁₇ : 17w² = 17² + 12·17z² − 32z⁴ に ℚ₁₇-点があると仮定する。ord₁₇(17w²) は奇数、ord₁₇(32z⁴) は偶数であることに注意して 付値を比較すると z, w ∈ ℤ₁₇ が必要。すると方程式から順に z ≡ 0, w ≡ 0 (mod 17)、最後に 17² ≡ 0 (mod 17³) となり矛盾。よって C₁₇(ℚ₁₇) = ∅。
  • 残りは群構造で自動処理：S(φ) は ℚ(S,2) の部分群で、 {1, 2, −2, −1} を含み 17 を含まない。±17, ±34 のどれかが入ると（2, −1 との積で）17 が入ってしまう。 よって S(φ)(E/ℚ) = {±1, ±2} ≅ (ℤ/2)²。

さらに、S(φ) の4元がすべて E′(ℚ) の点（O, (0,0), (8,−32), それらの和）から来ることが 分かったので、完全列 (X.4.2a) より Ш(E/ℚ)[φ] = 0。運が良い場合です。

φ̂ 側（E′ から E へ）

  • C′d : dw² = d² − 24dz² + 272z⁴（a′ = 12, b′ = −32, a′²−4b′ = 272）。
  • (0,0) ∈ E(ℚ) の像：δ(0,0) = 272 ≡ 17。よって 17 ∈ S(φ̂)。
  • d < 0 は実数で死ぬ：d < 0 なら左辺 dw² ≤ 0、 右辺 = d² − 24dz² + 272z⁴ は各項 ≥ 0 で d² > 0 なので正。C′d(ℝ) = ∅。
  • d = 2 は2進で死ぬ：C′₂ を z = Z/2 で整理すると 2w² = 4 − 12Z² + 17Z⁴。 ℚ₂-点があるとすると Z, w ∈ ℤ₂ が必要で、mod 2 で順に Z ≡ 0、w ≡ 0、 最後に 4 ≡ 0 (mod 8) となり矛盾。
  • まとめると S(φ̂)(E′/ℚ) = {1, 17} ≅ ℤ/2、しかも両方有理点由来なので Ш(E′/ℚ)[φ̂] = 0。

組み立て

E′(ℚ)/φ(E(ℚ)) ≅ (ℤ/2)²　（生成：(0,0), (8,−32)）
E(ℚ)/φ̂(E′(ℚ)) ≅ ℤ/2　（生成：(0,0)）
X.4.7 の完全列に代入 ⇒ E(ℚ)/2E(ℚ) ≅ (ℤ/2)²

E(ℚ)[2] = {O, (0,0)}（x² − 6x + 17 は判別式負で有理根なし）なので dim₂ E(ℚ)/2E(ℚ) = rank + dim₂ E(ℚ)[2] より rank = 1。結論：

E(ℚ) ≅ ℤ × ℤ/2ℤ
無限位数の点の例：(4, 6) ∈ E(ℚ)（64 − 96 + 68 = 36 = 6²）

これが「Mordell–Weil群を計算する」ことの全工程です。 弱MW（第2部）が保証した有限性を、Selmer群という計算可能な入れ物で実際に決定し、 降下定理（第4部）が「あとは生成元探しだけ」と言ってくれる。理論の全部品が動いています。

Section 07

Ш は本当に自明でないのか ─ Hasse原理の破れ AEC X.4.11

ここまでの例はすべて「運が良く」Ш[φ] = 0 でした。しかし Ш は一般に消えません。 歴史上最初の例は Lind（1940）と Reichardt（1942）によるものです：

C : 2w² = 1 − 17z⁴
すべての ℚp と ℝ で点を持つが、ℚ-点を持たない（練習問題5で証明する）

この C は然るべき楕円曲線の等質空間であり、Ш の非自明な元（位数2）を与えます。 少し後に Selmer（1951）は 3次曲線

3x³ + 4y³ + 5z³ = 0

が同種の例（x³ + y³ + 60z³ = 0 の Jacobian に対する位数3の Ш 元）であることを示し、 大規模な系列を研究しました。Selmer 群の名はこの研究に由来します。

Ш をどこまで攻められるか AEC X.4.12–4.14

  • 精密化された降下AEC X.4.12： S(mⁿ) の像 S(m,n) ⊂ S(m) を使うと 0 → E(K)/mE(K) → S(m,n) → mn−1Ш[mⁿ] → 0。 n を上げつつ有理点探索と並走させれば、Ш に「無限に m 割り可能な元」がない限りいつか階数が確定する。
  • Ш有限性予想AEC X.4.13：Ш(E/K) は有限だろう、という予想。 Kolyvagin・Rubin により特定の場合（解析階数 ≤ 1 など）には証明済み。 有限性が分かれば上の手続きは必ず停止する＝Mordell–Weil群の計算は完全アルゴリズムになる。
  • Casselsの対AEC X.4.14：交代双線形対 Γ : Ш × Ш → ℚ/ℤ が存在し、核は可除元のみ。とくに Ш が有限ならその位数は完全平方数。 「まだ有限かどうかも分からない群の位数について、これほど美しい制約が証明できる」という驚くべき定理。
? BSD 予想への一言（シリーズの出口）

Birch–Swinnerton-Dyer 予想AEC C.16.5は、階数 r と Ш を L関数 L(E,s) の s = 1 での挙動に結びつけます：ords=1 L(E,s) = r、 そして先頭係数の公式に #Ш とレギュレータ（第4部 §9）が現れる。 このシリーズで学んだ対象たち──rank、ĥ、Ш──が全部一つの解析的公式に集まるのが現代数論の風景です。

Section 08 ─ 展望(要約)

Ch. IX 整数点 ／ Ch. X の残り

Ch. IX：整数点（Siegelの定理）

Mordell–Weil が「有理点」の構造定理だとすると、Ch. IX は「整数点」の有限性定理です。 主結果は Siegel の定理：Weierstrass 方程式の S-整数点は有限個AEC IX §3–§4。 証明はディオファントス近似（Roth の定理AEC IX §1, §8）と、 なんとMordell–Weil定理そのものを使います（点を m で割って近似の質を上げる、descent の親戚）。 Baker の方法による有効版AEC IX §5、 「S外で良い還元を持つ楕円曲線は有限個」という Shafarevich の定理AEC IX.6.1も この章にあります。

⏩ 位置づけだけ知っていればよい

Ch. IX は VIII の応用編で、X §4 とは独立に読めます。 本シリーズの目的（VIII の理解）には「Mordell–Weil が近似定理と組み合わさって整数点の有限性を出す」 という一文の理解で十分です。

Ch. X §5–§6：twist の具体論

§5 は楕円曲線としての twist（2次twist E(d) : dy² = x³ + Ax + B など）の一般論 AEC X.5.4、§6 はそれを使って曲線族 Y² = X³ + DX の Mordell–Weil 群を D ごとに解析する大規模な実戦例ですAEC X §6。 §4 まで理解していれば、§6 は「腕試しの演習場」として最高の教材です。

Exercises

練習問題

問 1★ 基礎

基本図式 (∗∗) から完全列 0 → E′(K)/φ(E(K)) → S(φ) → Ш[φ] → 0（X.4.2a）を導く 図式追跡を、本文を見ずに再構成せよ。とくに「全射性」の議論で大域列 (∗) の右端の全射性を どこで使うかを明示すること。

解答を開く

本文 Section 04 の4段落（像が入る／写像が定義できる／全射／中央での完全性）を再構成すればよい。 全射性：c ∈ Ш[φ] ⊂ WC(E/K)[φ] に対し、(∗) の H¹(G,E[φ]) → WC(E/K)[φ] が全射である ことから持ち上げ ξ を取る。この全射性は長完全列で WC[φ] の次の項が H¹(G,E) 自身の φ 倍への 移行部分になっていること（Kummer型完全列の切り出し）による。 持ち上げの局所像が 0 になるのは c ∈ Ш だから。

問 2★ 基礎

本文 Section 06 の C₁₇(ℚ₁₇) = ∅ の議論（17進付値の偶奇）を、すべての行間を埋めて書き下せ。 すなわち：C₁₇ : 17w² = 17² + 12·17z² − 32z⁴ に (z,w) ∈ ℚ₁₇² があると仮定し、 (i) z, w ∈ ℤ₁₇ が必要であること、(ii) z ≡ 0 (mod 17)、(iii) w ≡ 0 (mod 17)、 (iv) 矛盾、の順で示せ。

解答を開く

(i) ord(z) < 0 とすると右辺の付値は 4·ord(z)（−32z⁴ の項が支配、ord₁₇(32)=0）で偶数、 左辺は 1 + 2·ord(w) で奇数、矛盾。よって z ∈ ℤ₁₇。すると右辺の付値 ≥ 0 は明らかで、 左辺 1 + 2ord(w) ≥ 0 から ord(w) ≥ 0（付値は整数なので ≥ 0）、w ∈ ℤ₁₇。 (ii) mod 17 で 0 ≡ −32z⁴、32 は単数なので z ≡ 0 (mod 17)。 (iii) z = 17z₁ を代入：17w² = 17² + 12·17³z₁² − 32·17⁴z₁⁴。17で割って w² = 17 + 12·17²z₁² − 32·17³z₁⁴ ≡ 0 (mod 17)、よって w ≡ 0 (mod 17)。 (iv) w = 17w₁ を代入し 17² で割ると 17w₁² = 1 + 12·17z₁²·… の形になり mod 17 で 0 ≡ 1、矛盾。（書き方は一通りでないが、付値の偶奇と mod 17 の交互使用が骨格。）

問 3★★ 標準

本文 Section 06 で「残りは群構造で自動処理」とした部分を厳密に実行せよ。 すなわち S(φ)(E/ℚ) が ℚ(S,2) = {±1,±2,±17,±34} の部分群であり {1,−1,2,−2} を含み 17 を含まないことから、S(φ) = {±1,±2} を結論せよ。 また、なぜ S(φ) が部分群になるのか（コホモロジー的理由）を一行で述べよ。

解答を開く

S(φ) は群準同型 H¹(G,E[φ]) → ∏vWC(E/Kv) の核なので部分群 （これが一行の理由。等質空間の言葉だけだと積構造は見えにくいが、コホモロジーでは自明になる）。 {±1, ±2} ⊂ S(φ) は本文の通り（−2 と 2 が入り、積で −1、1 は単位元）。 もし 17 ∈ S(φ) なら仮定に反する。34 ∈ S(φ) なら 34·2 = 68 ≡ 17（mod 平方） が入り矛盾。−17 なら −17·(−1) = 17 が矛盾。−34 なら −34·(−2) = 68 ≡ 17 が矛盾。 よって {±17, ±34} は全部除外され、S(φ) = {±1, ±2}。 位数8の群の部分群として位数4は正当（Lagrange）である。

問 4★★ 標準

回収装置 ψ の検算：Section 06 の C₂ の有理点 (z,w) = (1/2, 2) に対し ψ(z,w) = (d/z², −dw/z³)（d = 2）が (8, −32) を与えること、 この点が実際に E′ : Y² = X³ + 12X² − 32X の上にあること、 さらに δ(8,−32) = 8 ≡ 2 (mod (ℚ×)²) を確かめよ。

解答を開く

ψ(1/2, 2) = ( 2/(1/4), −2·2/(1/8) ) = (8, −32)。 曲線上にあること：8³ + 12·8² − 32·8 = 512 + 768 − 256 = 1024 = (−32)²。✓ δ の公式（X.4.9）は δ((X,Y)) = X（X ≠ 0）なので δ(8,−32) = 8 = 2·4 ≡ 2。✓ 「局所条件を満たす d に対して大域点が見つかれば、その d は確かに有理点由来」という Selmer 群の哲学が、この小さな計算に凝縮されている。

問 5★★★ 発展

（Lind–Reichardt：Hasse原理の破れの証明AEC X.4.11） C : 2w² = 1 − 17z⁴ について次を示せ。

(1) C(ℝ) ≠ ∅ と C(ℚ₁₇) ≠ ∅。
(2) C(ℚ₂) ≠ ∅ と、p ∤ 2·17 での C(ℚp) ≠ ∅。 （必要なら「z = ∞ 側のチャート」 2w′² = z′⁴ − 17（z′ = 1/z, w′ = w/z²）を使ってよい。）
(3) C(ℚ) = ∅。（ヒント：2u² = t⁴ − 17s⁴（整数、gcd(t,s)=1）に帰着させ、 u の素因数 p に対して 17 が mod p で平方剰余であることを示し、 平方剰余の相互法則で (p/17) = 1 を導き、u 全体が mod 17 で平方剰余、 したがって 2 が mod 17 で4乗剰余であることを結論して矛盾を出す。）

解答の方針を開く

(1) ℝ：|z| が小さければ 1 − 17z⁴ > 0 なので w が取れる。 ℚ₁₇：z = 0 とすると w² = 1/2。1/2 ≡ 9 (mod 17) は平方剰余（9 = 3²）なので Henselの補題で w ∈ ℤ₁₇ に持ち上がる。

(2) ℚ₂：z = 1/11 ∈ ℤ₂（11 は 2進単数）を試すと 1 − 17/11⁴ = 14624/14641、 w² = 7312/11⁴ = 16·457/11⁴。457 ≡ 1 (mod 8) なので 457 は ℤ₂ の平方数、 よって w = 4√457/121 ∈ ℚ₂ が存在する。 p ∤ 2·17：C の還元は 𝔽p 上の種数1の滑らかな曲線（2つのチャートを貼った射影モデルで考える）。 Hasse–Weil の評価 #C(𝔽p) ≥ p + 1 − 2√p > 0 により 𝔽p-点があり、 滑らかさから Hensel で持ち上がる。小さい p は手でも確認できる （例：p = 5 では z-チャートに解がないが、z′-チャートに (z′,w′) = (0, 2)：2·4 = 8 ≡ −17 ≡ 3 …となるのは −2 ≡ 3 の確認。両チャートを見る必要があるのがこの問題の教育的な罠）。

(3) 有理点 (z,w) があるとし、z = s/t（既約）とおいて分母を払うと 2u² = t⁴ − 17s⁴（u = wt² ∈ ℤ としてよい。u > 0 に取れる：u = 0 なら 17 が4乗数で矛盾）。 gcd(t,s) = 1。17 ∤ u：もし 17 | u なら t⁴ ≡ 0 (mod 17) で 17 | t、すると 17⁴ | 17s⁴ から 17 | s となり gcd に反する。 u の任意の奇素因数 p（p ≠ 17）：t⁴ ≡ 17s⁴ (mod p) で p ∤ s（さもなくば p | t）なので 17 ≡ (t/s)⁴：17 は mod p で4乗剰余、とくに平方剰余。17 ≡ 1 (mod 4) なので相互法則から (p/17) = (17/p) = 1。また (2/17) = 1（6² = 36 ≡ 2）、(−1/17) = 1（17 ≡ 1 mod 4）。 よって u = ±2e·∏pi の各因子が mod 17 で平方剰余、u 自身も平方剰余： u ≡ g2c（g は mod 17 の原始根）。 すると 2u² ≡ t⁴ (mod 17) より 2 ≡ t⁴·u⁻² = （4乗剰余）·（4乗剰余）⁻¹ は4乗剰余。 しかし mod 17 の4乗剰余は {1, 4, 13, 16} であり 2 は含まれない（2 の位数は 8）。矛盾。

まとめ：C はすべての局所体で点を持つが ℚ-点を持たない。 C が対応する楕円曲線（Jacobian）の Ш に、位数2の非自明元を与える。

問 6★★★ 発展

（階数の上界としての Selmer 群）φ = [2] の完全列 0 → E(ℚ)/2E(ℚ) → S(2)(E/ℚ) → Ш(E/ℚ)[2] → 0 から、ℤ/2-次元の等式

dim₂ S(2) = rank E(ℚ) + dim₂ E(ℚ)[2] + dim₂ Ш[2]

を導け。さらに Cassels の対（X.4.14）を認めて、「Ш が有限なら #Ш は完全平方数」を導け （ヒント：非退化な交代形式を持つ有限アーベル群の位数を考える）。

解答の方針を開く

前半：E(ℚ) ≅ ℤʳ × T（T 有限）より E(ℚ)/2E(ℚ) ≅ (ℤ/2)ʳ × T/2T、 そして dim₂ T/2T = dim₂ T[2]（有限アーベル群では 2T の余核と 2-捩れの次元が等しい： 構造定理で巡回成分ごとに確認）。よって dim₂ E(ℚ)/2E(ℚ) = r + dim₂ E(ℚ)[2]。 完全列の次元勘定でそのまま主張の等式になる。 この等式が「Selmer群 ＝ 階数の計算可能な上界」の正確な意味である： dim₂ S(2) − dim₂ E(ℚ)[2] ≥ r で、等号成立 ⟺ Ш[2] = 0。

後半：Ш 有限とすると Cassels の対 Γ は Ш 上非退化（可除元は有限群では 0 のみ）。 非退化な交代双線形形式 Γ : A × A → ℚ/ℤ を持つ有限アーベル群 A は、 シンプレクティック基底の構成（各巡回対 (x, y) with Γ(x,y) = 1/n を取り、直交補空間に降りる帰納法） により A ≅ B × B の形になる。よって #A = (#B)²。 p-成分ごとに同じ議論が通るので、各 p-部分の位数も平方数。
