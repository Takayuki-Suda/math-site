一次関数
y = ax + b

a と b を動かすと、グラフがどう変わるか体感しよう。

y = 1x + 0
a 傾き（グラフの角度） 1
b 切片（y軸との交点） 0
一次関数とは、x が1つ増えると y がある量だけ増える（または減る）関係のことです。

式 y = ax + b で表され、グラフにすると必ず直線になります。

たとえば「1分歩くと80m進む」なら、x分後の距離 y は y = 80x（a=80, b=0）と書けます。
a（傾き）は「x が 1 増えたとき、y がいくつ増えるか」を表します。

・a が大きい（正） → グラフが急な上がり坂
・a が小さい（正） → グラフがゆるやかな上がり坂
・a がマイナス → グラフが下がり坂
・a = 0 → 水平な直線（y = b）

スライダーで a を動かして確かめよう！
b（切片）は「x = 0 のとき y がいくつか」、つまりy 軸との交点を表します。

・b = 3 → グラフは (0, 3) を通る
・b = -2 → グラフは (0, -2) を通る
・b = 0 → グラフは原点 (0, 0) を通る

b を変えても傾きは変わらない（平行移動するだけ）。スライダーで確かめよう！
グラフから a と b を読み取る手順はこうです。

① グラフが y 軸と交わる点を見る → それが b
② 右に 1 進んだとき y がいくつ増えるかを数える → それが a

下の「2点の座標」を使って確かめてみよう。
グラフ上の2点
点 P（x = -3）
(-3, -3)
点 Q（x = 3）
(3, 3)
