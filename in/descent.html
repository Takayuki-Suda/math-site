Section 01

なぜ弱MWだけでは足りないのか AEC VIII §3 冒頭

まず反例で感覚を固定します。「A/mA 有限 ⇒ A 有限生成」は偽です。

  • A = ℝ：どんな m ≥ 1 でも ℝ/mℝ = 0（有限！）。だが ℝ は有限生成でない。
  • A = E(ℚp)：E(ℚp) は指数有限の部分群 ≅ ℤp を含むAEC VII.6.3。 ℤp/mℤp は有限（p ∤ m なら 0）なので E(ℚp)/mE(ℚp) は有限。 だが E(ℚp) は有限生成でない（ℤp は非可算）。

両方の反例に共通する病理は「m で無限回割れる元が大量にある」ことです （ℝ でも ℤp でも、任意の元が m で割れ続ける）。 数体上の楕円曲線ではこの病理が起きないことを言うために、「点の大きさ」＝高さを導入し、 「m倍写像は高さをほぼ m² 倍にする」ことを示します。 大きさが有界な点は有限個しかないので、無限に割り続けることは不可能になる──これが降下法の心です。

Section 02

降下定理 Descent Theorem

定理（降下定理） AEC VIII.3.1

A をアーベル群とし、関数 h : A → ℝ（高さ関数）が次の3性質を持つとする。

(i)　各 Q ∈ A に対し定数 C₁ = C₁(Q) があり：h(P + Q) ≤ 2h(P) + C₁　（∀P）
(ii)　ある整数 m ≥ 2 と定数 C₂ があり：h(mP) ≥ m²h(P) − C₂　（∀P）
(iii)　任意の定数 C₃ に対し { P ∈ A : h(P) ≤ C₃ } は有限集合

さらに (ii) の m について A/mA が有限ならば、A は有限生成である。

証明（完全版・コメント付き）

A/mA の剰余類の代表 Q₁, …, Qr を取ります（有限性の仮定）。任意の P ∈ A から出発して、 「m で割って余りを引く」操作を繰り返します：

P = mP₁ + Qi₁,　P₁ = mP₂ + Qi₂,　…,　Pn−1 = mPn + Qiₙ

各ステップで高さがどう変わるかを評価します。任意の j で：

h(Pj) ≤ (1/m²)( h(mPj) + C₂ )　　　（(ii) より）
　　　 = (1/m²)( h(Pj−1 − Qij) + C₂ )
　　　 ≤ (1/m²)( 2h(Pj−1) + C₁′ + C₂ )　（(i) を Q = −Qi に適用）
C₁′ = max{ C₁(−Qi) : i = 1,…,r }。C₁′ も C₂ も P に依らない定数

つまり1ステップごとに高さは「約 2/m² 倍＋定数」になります。m ≥ 2 なら 2/m² ≤ 1/2 なので、 これは縮小写像です。繰り返し使うと：

h(Pn) ≤ (2/m²)ⁿ h(P) + ( 1/m² + 2/m⁴ + ⋯ + 2ⁿ⁻¹/m²ⁿ )(C₁′ + C₂)
　　　 < (2/m²)ⁿ h(P) + (C₁′ + C₂)/(m² − 2)
　　　 ≤ 2⁻ⁿ h(P) + (C₁′ + C₂)/2　　（m ≥ 2 のとき）

n を十分大きく取れば h(Pn) ≤ 1 + (C₁′ + C₂)/2。一方、割り算の列を逆にたどると

P = mⁿPn + Σj=1n mj−1 Qij

なので、P は次の集合の元の一次結合です：

{ Q₁, …, Qr } ∪ { Q ∈ A : h(Q) ≤ 1 + (C₁′ + C₂)/2 }

(iii) よりこれは有限集合。よって A は有限生成です。

∎

高さ降下シミュレータ

証明中の縮小の様子を目で見てみます。棒グラフは上の不等式が保証する h(Pj) の上界です。 初期高さや定数をどれだけ大きくしても、数ステップで「床」（＝ 1 + (C₁′+C₂)/2 のライン）に落ちる ことを確認してください。

? Fermat の無限降下との関係

Fermat は「解があれば、より小さい解が作れる → 無限に小さくなれないから矛盾」という形で使いました（非存在の証明）。 Mordell–Weil では同じ縮小構造を「どんな点も、有限ステップで床に落ちる → 床の点と余り Qi で全部生成できる」 という生成の証明に転用しています。「無限降下」が非存在ではなく有限生成を生む──ここが歴史的にも面白い転回点です。

効果性の注意 AEC VIII.3.2

降下定理を「アルゴリズム」にするには、(1) 各 C₁(Qi)、(2) C₂、(3) 床以下の点の列挙、 の3つが計算可能であればよい。楕円曲線の高さ（次節）ではこれらすべてが有効に計算できます。 つまり、E(K)/mE(K) の生成元さえ手に入れば E(K) の生成元は必ず計算できる。 Mordell–Weil 定理の非有効性は、第2部・注意2で見た一点（E(K)/mE(K) の決定）に本当に集中しているのです。

Section 03

ℚ 上の高さ関数 ─ 定義と性質 (a)

あとは E(ℚ) の上に (i)(ii)(iii) を満たす関数を作れば、m = 2 の弱MW（第2部）とあわせて E(ℚ) の有限生成が出ます。E : y² = x³ + Ax + B（A, B ∈ ℤ）を固定します。

定義（素朴な高さ） AEC VIII §4

t = p/q ∈ ℚ（既約分数）に対し H(t) = max{ |p|, |q| }。そして

hx : E(ℚ) → ℝ,　hx(P) = log H(x(P))　（P ≠ O）,　hx(O) = 0

「x座標の分子・分母の大きい方の桁数」が点の大きさ、という素朴な定義である。hx ≥ 0 に注意。

性質 (iii) は明らかです：H(t) ≤ C なる有理数は分子分母が [−C, C] の整数なので高々 (2C+1)² 個、 各 x に対して y は高々2つAEC VIII.4.2(c)。 問題は (i)(ii)、つまり「加法・倍加が高さをどう変えるか」を加法公式から読み取ることです。

補題（高さの性質） AEC VIII.4.2
(a)　hx(P + P₀) ≤ 2hx(P) + C₁(P₀, A, B)
(b)　hx([2]P) ≥ 4hx(P) − C₂(A, B)
(c)　{ P : hx(P) ≤ C₃ } は有限
(a) の証明（要点＋行間）

P₀ = O や P ∈ {O, ±P₀} の場合は C₁ を大きく取れば済むので、それ以外とします。 既約分数表示で P = (a/d², b/d³)、P₀ = (a₀/d₀², b₀/d₀³) と書けます （Weierstrass方程式の分母は必ずこの形になります：x = u/v が既約なら y² の分母から v = d² が従う）。 加法公式AEC III.2.3(d)を曲線の式で整理すると：

x(P + P₀) = ( (xx₀ + A)(x + x₀) + 2B − 2yy₀ ) / (x − x₀)²
　　　　 = ( (aa₀ + Ad²d₀²)(ad₀² + a₀d²) + 2Bd⁴d₀⁴ − 2bd·b₀d₀ ) / (ad₀² − a₀d²)²

分子・分母は a, d², bd の「2次以下」の式です。約分は高さを下げるだけなので、

H(x(P+P₀)) ≤ C′ · max{ |a|², |d|⁴, |bd| }

となります（C′ は A, B, a₀, b₀, d₀ だけの定数）。 H(x(P)) = max{|a|, |d|²} なので、|a|² と |d|⁴ はちょうど H(x(P))² の形。 唯一の邪魔者は |bd| です。ここで「P が曲線上にある」ことを使います： b² = a³ + Aad⁴ + Bd⁶ より

|b| ≤ C″ max{ |a|³ᐟ², |d|³ }　⇒　|bd| ≤ C″ max{ |a|³ᐟ²|d|, |d|⁴ } ≤ C″ max{ |a|², |d|⁴ }
最後の不等号：|a|¹ᐟ² ≥ |d| なら |a|³ᐟ²|d| ≤ |a|²、そうでなければ |a|³ᐟ²|d| < |d|⁴

よって H(x(P+P₀)) ≤ C₁ max{|a|,|d|²}² = C₁ H(x(P))²。対数を取れば (a) が出ます。

∎
? (a) の教訓

加法公式は x について次数2の有理式なので「高さ2倍＋定数」はほぼ自明── ただし y の項（|bd|）が混ざるところだけは、曲線の方程式そのもので y を x で抑える必要がある。 「点が曲線に乗っている」という情報が定量評価に化ける瞬間です。

Section 04

急所：倍加は高さを4倍にする ─ 性質 (b)

(b) は (a) より難しい。理由は不等式の向きです。 (a) は「大きくなりすぎない」（上から抑える）だけなので約分を無視できましたが、 (b) は「小さくなりすぎない」（下から抑える）ので、 分数がどれだけ約分されうるかを制御しなければなりません。

倍加公式を斉次化します。x = x(P) = p/q（既約）として：

x([2]P) = F(p, q) / G(p, q)
F(X, Z) = X⁴ − 2AX²Z² − 8BXZ³ + A²Z⁴
G(X, Z) = 4X³Z + 4AXZ³ + 4BZ⁴　（= 4y²·Z⁴ に対応）

F(X,1) と G(X,1) は互いに素な多項式（Δ ≠ 0 のとき）なので、終結式の理論により 「互いに素さの証明書」となる恒等式が存在します。AECはそれを明示的に書き下しています：

副補題（明示的な恒等式） AEC VIII.4.3

Δ' = 4A³ + 27B²（判別式の −16 倍を除いた部分）とおくと、ℤ[A,B,X,Z] の恒等式として

f₁(X,Z)·F(X,Z) − g₁(X,Z)·G(X,Z) = 4Δ'·Z⁷
f₂(X,Z)·F(X,Z) − g₂(X,Z)·G(X,Z) = 4Δ'·X⁷
f₁ = 12X²Z + 16AZ³,　g₁ = 3X³ − 5AXZ² − 27BZ³　（f₂, g₂ も次数3の明示的な多項式）
(b) の証明（完全版・コメント付き）

C₂ ≥ 4·max{ hx(T) : T ∈ E(ℚ)[2] } と取っておけば [2]P = O の場合は自明なので、[2]P ≠ O とします。 約分量を δ = gcd( F(p,q), G(p,q) ) とおきます。恒等式に (X,Z) = (p,q) を代入すると：

f₁(p,q)F(p,q) − g₁(p,q)G(p,q) = 4Δ'q⁷,　f₂(p,q)F(p,q) − g₂(p,q)G(p,q) = 4Δ'p⁷

δ は左辺の両方を割るので、4Δ'q⁷ と 4Δ'p⁷ の両方を割ります。gcd(p,q) = 1 なので δ | 4Δ'。つまり約分は高々 |4Δ'|（P に依らない定数！）しか起きません。よって：

H(x([2]P)) ≥ max{ |F(p,q)|, |G(p,q)| } / |4Δ'|

あとは max{|F|, |G|} を下から max{|p|,|q|}⁴ で抑えれば終わりです。再び恒等式から：

|4Δ'q⁷| ≤ 2 max{|f₁|, |g₁|} · max{|F|, |G|},　|4Δ'p⁷| ≤ 2 max{|f₂|, |g₂|} · max{|F|, |G|}

f₁, g₁, f₂, g₂ は次数3の多項式なので max{|f₁|,…,|g₂|} ≤ C·max{|p|,|q|}³。まとめると

max{ |4Δ'p⁷|, |4Δ'q⁷| } ≤ 2C·max{|p|,|q|}³ · max{|F|, |G|}
⇒　max{|F|, |G|} / |4Δ'| ≥ (2C)⁻¹ max{|p|,|q|}⁴ = (2C)⁻¹ H(x(P))⁴

対数を取って hx([2]P) ≥ 4hx(P) − C₂ を得ます。

∎
? なぜ「4倍」は崩れないのか ── 議論の本質

倍加公式は次数4の有理写像なので、高さが「最大4倍」なのは当たり前。 非自明なのは「ほぼきっかり4倍」が下からも保証されることで、 その理由は F と G が共通零点を持たない（＝曲線が非特異、Δ ≠ 0）ことにあります。 共通零点がないことの定量版が副補題の恒等式で、「約分 δ は 4Δ' を割る」という形で 幾何の非退化性が数論の評価に翻訳されています。 この「射影空間の射は高さを次数倍する」という現象は、一般論として AEC VIII.5.6 に昇格します。

Section 05

組み立て：ℚ 上の Mordell–Weil 定理

定理 AEC VIII.4.1

E/ℚ を楕円曲線とすると、E(ℚ) は有限生成である。

証明

弱Mordell–Weil定理（第2部、m = 2）より E(ℚ)/2E(ℚ) は有限。 hx は VIII.4.2 の (a)(b)(c) により降下定理の仮定 (i)(ii)(iii)（m = 2）を満たす。 よって降下定理（VIII.3.1）から E(ℚ) は有限生成。

∎

注意：ここで使う弱MWは m = 2 の場合だけで、しかも E[2] ⊂ E(ℚ) は仮定していません （第2部の帰着補題が体拡大を処理してくれるため）。2つの部品が噛み合う様子を味わってください。

数値で「4倍＋O(1)」を体感する ─ E : y² = x³ + 3, P = (1, 2)

倍加公式（A = 0, B = 3）で実際に計算すると：

点x座標hx（=log H）前項との比
P1log 1 = 0─
2P−23/16log 23 ≈ 3.14─（h=0 なので比は無意味）
4P2540833/7744log 2540833 ≈ 14.7514.75/3.14 ≈ 4.7

検算のヒント：x(2P) = (x⁴ − 24x)/(4x³ + 12)、x = 1 で (1−24)/16 = −23/16。 次に F(−23,16) = (−23)⁴ − 24·(−23)·16³ = 2540833、G(−23,16) = 4(−23)³·16 + 12·16⁴ = 7744、 gcd = 1（4Δ' = 972 = 2²·3⁵ と 7744 = 2⁶·11² の共通因子は 4 だが 2540833 は奇数）。 比が 4 に落ち着いていく様子、そして「+O(1)」のぶんだけ 4 からずれる様子が見えます。 ちなみにこの数列から lim hx(2ⁿP)/4ⁿ ≈ 0.9 が読み取れ、これが次節の標準高さの値の近似です。

Section 06 ─ 展望（要約）

VIII §5–§11 を地図で歩く

§5–§6：一般の数体への拡張と「高さ機械」

一般の数体 K では、すべての素点（絶対値）をまとめて高さを定義します： P = [x₀ : … : xN] ∈ ℙᴺ(K) に対し HK(P) = Πv maxi|xi|vnv。 積公式のおかげで座標の取り方に依りません。重要な成果は2つ：

  • Northcott の有限性AEC VIII.5.11：高さと次数が有界な点は有限個。性質(iii)の一般版。
  • 高さ機械：偶関数 f ∈ K(E) に対し hf が「ほぼ2次形式」になる （hf(P+Q) + hf(P−Q) = 2hf(P) + 2hf(Q) + O(1)、平行四辺形則）AEC VIII.6.2。 これから (i)(ii) 型の不等式が全て従い、一般の数体上の Mordell–Weil 定理 AEC VIII.6.7 が §4 と同じ骨格で証明されます。
⏩ ここで止まらない

§5–§6 は「§4 で手計算したことを、任意の数体・任意の射影埋め込みで通用する形に鍛え直す」章です。 ℚ の場合の骨格（今日の内容）を理解していれば、§5–§6 は「同じ議論の一般化」として安心して読み流せます。 深入りは標準高さ（§9）が必要になったときで十分です。

§7：捩れ部分群 ─ Lutz–Nagell と Mazur

定理（Lutz–Nagell） AEC VIII.7.1–7.2

E : y² = x³ + Ax + B（A,B ∈ ℤ）、P = (x,y) を捩れ点（≠ O）とすると、 x, y ∈ ℤ であり、y = 0 または y² | 4A³ + 27B²。

これで E(ℚ)tors は有限回の割り算チェックで決定できます（練習問題5）。 さらに深い事実として、Mazur の定理AEC VIII.7.5：E(ℚ)tors は ℤ/Nℤ（N ≤ 10 または 12）か ℤ/2ℤ × ℤ/2Nℤ（N ≤ 4）の15種類に限る。 （証明はAECの範囲外。主張だけ知っていれば十分です。）

§8：最小判別式・大域最小モデル

K = ℚ では全素点で同時に最小な Weierstrass 方程式（大域最小モデル）が取れますAEC VIII.8.2–8.3 （一般の数体では類数1なら取れる）。第2部で S を決めるときの「最小方程式」の裏付けです。要約でOK。

§9：標準高さ（Néron–Tate height）

hx の「+O(1)」の揺らぎを、極限で完全に消すことができます：

ĥ(P) = (1/2)·limn→∞ 4⁻ⁿ hx([2ⁿ]P)
（正規化は流儀により1/2倍の差がある。AEC VIII §9 は h_f/deg f の極限で定義）
  • ĥ は正確な2次形式になり、ĥ(P) = 0 ⟺ P は捩れ点AEC VIII.9.3。
  • E(K) ⊗ ℝ 上で正定値AEC VIII.9.6。 双線形形式 ⟨P,Q⟩ = ĥ(P+Q) − ĥ(P) − ĥ(Q) の行列式が楕円レギュレータAEC VIII.9.7で、 BSD予想の公式に登場します。
  • 極限の存在証明は「Tateの望遠鏡トリック」と呼ばれる初等的な議論です（練習問題6で実際にやります）。

§10–§11：階数の謎・Szpiro予想（読み物として）

階数 r には、いまだに基本的な未解決問題が並びます：階数は非有界かAEC VIII.10.1（予想）、 階数の「平均」はいくつか、など。§11 は Szpiro 予想（判別式と導手の不等式）と ABC 予想の同値性を扱う読み物で、 Mordell–Weil の理解には不要ですAEC VIII §11。

Exercises

練習問題

問 1★ 基礎

降下定理の証明で「n を十分大きく取れば h(Pn) ≤ 1 + (C₁′+C₂)/2」とした。 h(P) と C₁′+C₂ が与えられたとき、これを保証する n を明示的に1つ与えよ（m = 2 とする）。

解答を開く

評価式は h(Pn) ≤ 2⁻ⁿ h(P) + (C₁′+C₂)/2 なので、2⁻ⁿ h(P) ≤ 1 となれば十分。 すなわち n ≥ log₂ h(P) なる任意の整数、たとえば n = max{0, ⌈log₂ h(P)⌉} が取れる。 高さの対数（≒桁数の対数）ステップで床に到達する──降下が「速い」ことも読み取れる。

問 2★ 基礎

高さの性質 (iii)（有界集合の有限性）だけを使って、E(ℚ) の捩れ部分群が有限であることを 降下定理を経由せずに直接示せ。（ヒント：捩れ点の高さは有界であることを (a)(b) から導く。）

解答を開く

P を捩れ点、位数を k とする。(b) を繰り返すと hx(2ⁿP) ≥ 4ⁿ hx(P) − C₂(1 + 4 + … + 4ⁿ⁻¹) ≥ 4ⁿ( hx(P) − C₂/3 )。もし hx(P) > C₂/3 なら左辺は n とともに発散するが、 {2ⁿP} は有限集合（P の生成する巡回群に含まれる）なので高さは有界、矛盾。 よってすべての捩れ点は hx ≤ C₂/3 を満たし、(iii) より捩れ点は有限個。 ──これは「ĥ(捩れ点) = 0」（VIII.9.3）の素朴版である。

問 3★★ 標準

E : y² = x³ + 3、P = (1,2) について、本文の表の 4P の行を検算せよ。 すなわち F(−23,16) = 2540833、G(−23,16) = 7744 を計算し、 gcd(2540833, 7744) = 1 を「δ | 4Δ' = 972」を使って確認し、hx(4P) = log 2540833 を結論せよ。

解答を開く

F(X,Z) = X⁴ − 24XZ³（A=0, B=3 なので −2AX²Z² と A²Z⁴ は消える）： F(−23,16) = 279841 + 24·23·4096 = 279841 + 2260992 = 2540833。 G(X,Z) = 4X³Z + 12Z⁴：G(−23,16) = −778688 + 786432 = 7744 = 2⁶·11²。 δ | 4Δ' = 972 = 2²·3⁵ かつ δ | 7744 = 2⁶·11² なので δ | 4。 F(−23,16) は奇数（偶数＋奇数）なので δ = 1。 よって既約分数として x(4P) = 2540833/7744、分子の方が大きいので hx(4P) = log 2540833 ≈ 14.75。

問 4★★ 標準

A = 0, B = 3 の場合に副補題（VIII.4.3）の第1恒等式 f₁F − g₁G = 4Δ'Z⁷ を実際に展開して検証せよ。 （f₁ = 12X²Z, g₁ = 3X³ − 81Z³, F = X⁴ − 24XZ³, G = 4X³Z + 12Z⁴, 4Δ' = 972。）

解答を開く

f₁F = 12X²Z(X⁴ − 24XZ³) = 12X⁶Z − 288X³Z⁴。
g₁G = (3X³ − 81Z³)(4X³Z + 12Z⁴) = 12X⁶Z + 36X³Z⁴ − 324X³Z⁴ − 972Z⁷ = 12X⁶Z − 288X³Z⁴ − 972Z⁷。
差を取ると f₁F − g₁G = 972Z⁷ = 4·243·Z⁷ = 4Δ'Z⁷（Δ' = 27·9 = 243）。✓ 一般の A, B での検証は「退屈だが確実にできる計算」（AECの言葉では tedious calculation）である。 f₁, g₁ を Euclid の互除法で自力導出してみるのもよい練習になる。

問 5★★ 標準

Lutz–Nagell（VIII.7.1–7.2）を使って E : y² = x³ + 3 の捩れ部分群が自明（= {O}）であることを示し、 P = (1,2) と Mordell–Weil 定理から rank E(ℚ) ≥ 1 を結論せよ。

解答を開く

4A³ + 27B² = 0 + 243 = 243 = 3⁵。捩れ点 (x,y) ≠ O は x,y ∈ ℤ で、 y = 0 なら x³ = −3（整数解なし）。y ≠ 0 なら y² | 243 より y² ∈ {1, 9, 81}、 つまり y ∈ {±1, ±3, ±9}。それぞれ x³ = y² − 3 ∈ {−2, 6, 78} となるが、 いずれも整数の立方でない。よって捩れ点は O のみ。 P = (1,2) ∈ E(ℚ) は O でないので無限位数。E(ℚ) ≅ ℤʳ（捩れなし）で r ≥ 1。 （実は r = 1 で P が生成元だが、その証明には第5部の降下計算が要る。）

問 6★★★ 発展

（Tateの望遠鏡トリック＝標準高さの存在AEC VIII.9.1 の核心） h = hx とする。まず (a)：ある定数 C で |h(2P) − 4h(P)| ≤ C（∀P）が成り立つことを、 VIII.4.2(b) と「上からの評価 h(2P) ≤ 4h(P) + C」（これは (a) と同様の粗い評価で出る）から認めてよい。 このとき (b)：数列 aₙ = 4⁻ⁿ h(2ⁿP) が Cauchy 列であることを示し、 極限 ĥ'(P) = lim aₙ が存在して |ĥ'(P) − h(P)| ≤ C/3 を満たすことを証明せよ。

解答の方針を開く

隣接項の差を評価する：|an+1 − an| = 4⁻⁽ⁿ⁺¹⁾|h(2·2ⁿP) − 4h(2ⁿP)| ≤ C·4⁻⁽ⁿ⁺¹⁾。 よって n < m に対し |am − an| ≤ C Σk>n 4⁻ᵏ = C·4⁻ⁿ/3 → 0、Cauchy列。 極限を ĥ'(P) とすると |ĥ'(P) − a₀| ≤ Σk≥1 C·4⁻ᵏ = C/3、a₀ = h(P) なので主張が従う。

補足：同じ望遠鏡評価から ĥ'(2P) = 4ĥ'(P) が誤差なしで成り立つことも直ちに出る （定義の極限をずらすだけ）。「+O(1) を極限で消して完全な2次形式にする」という §9 の設計思想が、 この計算に凝縮されている。平行四辺形則を使うと同様の議論で ĥ' が2次形式であることまで示せる （AEC VIII.9.3 の証明）。
