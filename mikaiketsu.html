楕円曲線 ── すべてはこの曲線から

たった一本の曲線が、358年の難問フェルマーの最終定理を打ち倒し、 今も100万ドルの未解決問題の中心に座っている。 それが楕円曲線だ。

下のスライダーで a と b を動かして、曲線の形が変わるのを確かめてみてほしい。

y² = x³ − 2x + 2

Δ ＝ −16(4a³ ＋ 27b²) ＝ 0 になると曲線が自分自身と交わって「特異」になる。 なめらかな曲線（Δ ≠ 0）だけが楕円曲線と呼ばれる。

Defeated 1995 ─ 撃破された壁
証明済み（1995）358年戦争

フェルマーの最終定理

xⁿ + yⁿ = zⁿ

「n ≥ 3 のとき自然数解はない。私は真に驚くべき証明を見つけたが、 この余白はそれを書くには狭すぎる」──1637年、フェルマーが本の余白に残した一行。

谷山豊と志村五郎の予想が楕円曲線とモジュラー形式を結びつけ、 アンドリュー・ワイルズが7年間屋根裏にこもって証明を完成させた。 壁は、問い続けた人間によってしか壊れない。

リーマン予想

ζ(s) = 0 ⇒ Re(s) = ½

素数は一見デタラメに現れる。だがその裏に完璧な秩序がある── ゼータ関数の零点はすべて一直線上に並ぶ、という1859年の予想。 これが証明されれば、素数の分布は完全に手の内に入る。165年以上、誰も倒せていない。

BSD予想（バーチ・スウィンナートン＝ダイアー予想）

y² = x³ + ax + b

楕円曲線の上に有理数の点（分数で書ける点）はいくつあるのか。 無限にあるのか、有限で尽きるのか──その答えが、 曲線ごとに決まる「L関数」の振る舞いに刻まれている、という予想だ。
