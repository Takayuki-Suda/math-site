行間ノート ─ 弱Mordell–Weil N2/N3

不分岐性の完全証明
Unramifiedness outside S, with every step justified

「L/K は S の外で不分岐」──本編 STEP 2 で最も行間が広いのはここです。 還元写像を原始座標から構成し、Galois 作用との両立性を手で証明し、 惰性群による不分岐判定を経由して、合成体の問題まで一気に片づけます。

TEXT: Silverman, AEC 2nd ed. ─ VIII.1.3–1.5（完全展開）

本編Ⅱ 弱MW N0 帰着補題 N1 Kummer対 N2 不分岐性 N3 体論の有限性
Section 01

このノートで埋める行間

  • ① S が有限集合である理由の1行ずつの根拠（「有限個を除いて係数が整」はなぜか）
  • ② 還元写像 P ↦ P̃ の構成そのもの（原始座標・well-defined 性）──本編では存在を認めていた
  • ③ 還元と Galois 作用の両立性 (Qσ)˜ = Q̃σ̃ の証明──本編の計算の心臓部なのに、通常どの本でも「明らか」で済まされる
  • ④「惰性群が自明に作用 ⟺ 不分岐」の正確な言明（前提として明示・出典付き）
  • ⑤ K(Q)/K が有限次であることの証明（そもそも「体拡大の分岐」を語る前提）
  • ⑥ L は無限個の生成体の合成かもしれない──その不分岐性をどう処理するか

N0 の P1–P4、N1 の Q1–Q4 は引き続き使用します。E[m] ⊂ E(K) の仮定は (a) でのみ使い、(b) では使いません（Section 08 で明記）。

Section 02

前提の固定（R1–R5）

代数的整数論からの入力を先に全部並べます。出典は Neukirch（付録 数論の武器庫の棚番号）と AEC です。

前提 R1 ─ 素点・完備化・分岐 ANT I §8, II §8

数体 M の有限素点 w には正規化付値 ordw : M× ↠ ℤ、完備化 Mw、 整数環と剰余体 kw（有限体、標数 pw）が付随する。 有限次拡大 M/K と w | v に対し分岐指数 e(w/v) が定まり、K× 上で ordw = e(w/v)・ordv。 「M/K が v で不分岐」とは、v の上の全ての w で e(w/v) = 1（剰余体拡大の分離性は有限体ゆえ自動）。 e は完備化で計算してもよい：e(w/v) = e(Mw/Kv)。

前提 R2 ─ K への延長・分解群・惰性群 ANT I §9 / II §9

v の K への延長 v̄ が存在する。分解群と惰性群を

Dv̄/v = { σ ∈ GK/K : v̄ ∘ σ = v̄ },　σ ∈ D は剰余体 k(v̄) に σ̃ を誘導
Iv̄/v = { σ ∈ Dv̄/v : σ̃ = idk(v̄) }

と定める。塔での関係：K ⊂ K′ ⊂ K、w′ = v̄|K′ のとき Dv̄/w′ = Dv̄/v ∩ GK/K′、I も同様 （定義から直ちに従う：条件「v̄∘σ = v̄」「σ̃ = id」は同じで、動く群だけが小さくなる）。

前提 R3 ─ 惰性群による不分岐判定 ANT I §9（惰性体）

Tv̄ := （Iv̄/v の固定体）とおく（惰性体）。有限次拡大 M/K（M ⊂ K）について：

M ⊂ Tv̄　⟺　M/K は素点 v̄|M で不分岐（e = 1）

さらに、M の v の上の任意の素点は適当な延長 v̄′ の制限 v̄′|M として得られるANT II.8.1。 よって「M/K が v で不分岐」⟺「v の全ての延長 v̄′ で M ⊂ Tv̄′」。

▸ なぜこの言い換えを使うのか（行間⑥の解消装置）

「Tv̄ に含まれるか」という包含条件に翻訳すると、 合成体の不分岐性が自明になります：M₁ ⊂ T かつ M₂ ⊂ T なら M₁M₂ ⊂ T（体の包含だから）。 部分拡大も自明：M′ ⊂ M ⊂ T なら M′ ⊂ T。 e の積公式で個別に戦うより、この一手が圧倒的に強い。N3 でも再利用します。

前提 R4 ─ 局所還元理論 AEC VII.2.1 / VII.3.1

F を数体の有限素点での完備化（局所体）、剰余体 k、標数 p とする。 E/F が F-整係数・単元判別式の Weierstrass 方程式（＝良い還元のモデル）で与えられているとき：

(i) 還元写像 red : E(F) → Ẽ(k) は群準同型であるAEC VII.2.1（良い還元なので E₀(F) = E(F)）。

(ii) p ∤ m ならば red の E(F)[m] への制限は単射であるAEC VII.3.1(b)。

(ii) の理由の骨子を開く（形式群）

核 E₁(F) = ker(red) は形式群 Ê(𝔪) と同型AEC VII.2.2。 形式群では、[m](T) = mT + （高次項）の1次の係数 m が p ∤ m により単元なので、 [m] : Ê(𝔪) → Ê(𝔪) は形式的逆関数定理により同型AEC IV.2.3。 特に Ê(𝔪) に m-捩れはない。よって E(F)[m] ∩ E₁(F) = {O}、 すなわち red を E(F)[m] に制限すると核が自明＝単射。

前提 R5 ─ 完備化との両立

数体 M と有限素点 w について E(M) ⊂ E(Mw) であり、 M の付値環 O(w) = {a ∈ M : ordw(a) ≥ 0} を使って定義した還元（Section 04）と、 Mw の整数環を使って定義した還元は E(M) 上で一致する。 根拠：O(w) は Mw の整数環と M の共通部分であり、剰余体は同一ANT II.4.3。 原始座標はどちらの環で見ても原始的である。

Section 03

S は有限集合である（行間①）

S = { v : E が v で悪い還元 } ∪ { v : v(m) ≠ 0 } ∪ MK∞
1

Weierstrass 方程式を1本固定する：係数 a₁, …, a₆ ∈ K、判別式 Δ ≠ 0。各係数 ai について { v ∈ MK0 : v(ai) < 0 } は有限。

▸ 根拠　a ∈ K× の分数イデアル (a) は有限個の素イデアルの積に一意分解するANT I §3。v(a) ≠ 0 となるのはその素因子に対応する v だけ。

2

{ v : v(Δ) ≠ 0 } も有限。{ v : v(m) ≠ 0 } も有限。MK∞ は高々 [K:ℚ] 個。

▸ 根拠　Step 1 と同じ（Δ, m ∈ K×）。無限素点の個数は実埋め込み＋複素埋め込み対の個数ANT I §5。

3

v ∉ S なる有限素点では、固定した方程式は v-整係数かつ v(Δ) = 0。したがって E は v で良い還元を持ち、この方程式自身が良いモデルである。

▸ 根拠　係数が整で判別式が単元なら、還元した方程式は kv 上の Weierstrass 方程式で判別式 Δ̃ = Δ mod 𝔪v ≠ 0、つまり非特異＝楕円曲線。「整係数・単元判別式のモデルが存在 ⟺ 良い還元」AEC VII.5.1(b)。（このモデルは v で最小でもある：整モデルの中で v(Δ) = 0 は最小値。）

4

拡大しても壊れない：K′/K 有限、w | v、v ∉ S ならば、同じ方程式は w-整係数かつ w(Δ) = 0。つまり E/K′ も w で良い還元を持ち、還元曲線は Ẽv の係数拡大である。

▸ 根拠　K の元 a に対し ordw(a) = e(w/v)・ordv(a)（R1）。ordv(ai) ≥ 0 ⇒ ordw(ai) ≥ 0、ordv(Δ) = 0 ⇒ ordw(Δ) = 0。分岐していても成り立つことに注意（e 倍しても 0 は 0）。

Section 04

還元写像の構成と Galois 両立性（行間②③）

補題 N2-A（還元写像の構成）

M を数体、w を有限素点、O(w) = {a ∈ M : ordw(a) ≥ 0}、剰余体 k。 このとき写像 red : ℙ²(M) → ℙ²(k) が次で well-defined に定まる： A = [x₀ : x₁ : x₂] の代表を mini ordw(xi) = 0 となるように取り（原始座標）、 red(A) := [x̄₀ : x̄₁ : x̄₂]。 さらに E が w-整係数・単元判別式の方程式で与えられていれば red(E(M)) ⊂ Ẽ(k)。

1

原始座標は存在する。

▸ 根拠　任意の代表 (x₀,x₁,x₂) ≠ 0 に対し n := mini ordw(xi) ∈ ℤ。ordw(π) = 1 なる π ∈ M が存在する（ordw は全射：R1）ので π−n 倍すれば min = 0 にできる。

2

原始座標は単元倍を除いて一意。したがって red(A) は代表の取り方に依らない。

▸ 根拠　2組の原始座標は λ ∈ M× 倍で移り合う（射影座標の定義）。両方の min が 0 なので ordw(λ) = 0、つまり λ は単元。単元の還元は k× の元なので、還元後も射影座標として同じ点。また min = 0 より少なくとも1つの x̄i ≠ 0、つまり (x̄) ≠ (0,0,0) で red(A) は本当に ℙ²(k) の点。

3

A ∈ E(M) ならば red(A) ∈ Ẽ(k)。

▸ 根拠　E の同次方程式 F(x₀,x₁,x₂) = 0 は O(w) 係数。環準同型 O(w) → k を通すと 0 = F(x)の還元 = F̃(x̄)。F̃ は Ẽ の定義方程式。

補題 N2-B（Galois 両立性）──行間③の核心

さらに M/K が Galois で、E は K 上定義され、σ ∈ GM/K が w ∘ σ = w を満たすとする。 σ は O(w) と極大イデアルを保つので剰余体に σ̃ : k → k を誘導する。このとき全ての A ∈ E(M) で

red(Aσ) = red(A)σ̃
4

A の原始座標 (x₀,x₁,x₂) を取ると、(x₀σ, x₁σ, x₂σ) は Aσ の原始座標である。

▸ 根拠　ordw(xiσ) = (ordw∘σ)(xi) = ordw(xi)（仮定 w∘σ = w）。よって min も 0 のまま。Aσ ∈ E(M) は E の係数が K の元で σ 不変だから（N0 の P2 と同じ理屈）。

5

red(Aσ) = [ (x₀σ)¯ : (x₁σ)¯ : (x₂σ)¯ ] = [ x̄₀σ̃ : x̄₁σ̃ : x̄₂σ̃ ] = red(A)σ̃。

▸ 根拠　最初の等号は Step 4（原始座標で還元を計算してよい：補題 N2-A の一意性）。2つ目は σ̃ の定義（σ̃(ā) := (σa)¯、これが well-defined なのは σ が極大イデアルを保つから）。

N2-A・N2-B ∎
? この2つの補題の意味

本編で1行だった「(Qσ − Q)˜ = Q̃σ − Q̃」は、実は N2-A（well-defined性）＋ N2-B（両立性）＋ R4(i)（準同型性）の3点セットです。 どの教科書でも「明らか」で通過する箇所ですが、原始座標まで降りれば全て単なる付値の計算です。

Section 05

主定理 (b) の証明 ─ L/K は S の外で不分岐

定理 AEC VIII.1.5(b)

L = K([m]−1E(K)) とすると、L/K は S の外で不分岐。

1

目標の言い換え：任意の v ∉ S と任意の延長 v̄ について「全ての Q ∈ [m]−1E(K) に対し K(Q) ⊂ Tv̄」を示せば十分。

▸ 根拠　これが言えれば L =（K(Q) たちの合成）⊂ Tv̄。すると L の任意の有限次部分拡大 M も M ⊂ Tv̄ なので、R3 により M は v̄|M で不分岐。v̄ が任意なので v の全素点で不分岐（R3 後半）、v ∉ S が任意なので「S 外不分岐」の定義（＝全ての有限次部分拡大が S 外不分岐）を満たす。行間⑥がここで消える。

2

K(Q)/K は有限次で、[K(Q) : K] ≤ m²（行間⑤）。

▸ 根拠　P := [m]Q ∈ E(K) のファイバー [m]−1(P) は { Q + T : T ∈ E[m] }、ちょうど m² 個（差を取れば ker[m] = E[m]：P1, P4）。任意の τ ∈ GK/K で [m](Qτ) = Pτ = P（P2）なので Qτ はこのファイバーの元。K-埋め込み K(Q) ↪ K は K の自己同型 τ に延長でき（Q4）、生成元 x(Q), y(Q) の行き先は Qτ で決まるから、埋め込みの個数 ≤ #ファイバー = m²。char 0 では次数 = 埋め込み数（分離性）。

3

v ∉ S、延長 v̄、σ ∈ Iv̄/v、Q ∈ [m]−1E(K) を任意に固定する。目標：Qσ = Q。M := K(Q) の Galois 閉包（⊂ K）とおくと、M/K は有限次 Galois で Q と Qσ の座標を含む。

▸ 根拠　Galois 閉包の存在と有限次性は体論（Step 2 で K(Q)/K 有限次）。Qσ の座標は x(Q), y(Q) の共役なので Galois 閉包に入る。

4

w := v̄|M とおく。E は w で良い還元を持ち、還元写像 red : E(Mw) → Ẽ(kw) は群準同型。E(M) 上では原始座標による還元（N2-A）と一致する。

▸ 根拠　良い還元：Section 03 Step 4（v ∉ S、w | v）。準同型性：R4(i)。一致：R5。

5

R := Qσ − Q とおくと R ∈ E(M)[m]。

▸ 根拠　Q, Qσ ∈ E(M)（Step 3）で E(M) は群（P1）。[m]R = [m](Qσ) − [m]Q = ([m]Q)σ − [m]Q（P2）= O、最後は [m]Q ∈ E(K) が σ で不動（P2）だから。ここで E[m] ⊂ E(K) は使っていない。

6

σ|M ∈ GM/K は w∘(σ|M) = w を満たし、誘導される σ̃ は kw 上恒等写像である。

▸ 根拠　σ(M) = M（M/K Galois・正規性）。w∘σ|M = (v̄∘σ)|M = v̄|M = w（σ ∈ I ⊂ D、R2）。剰余体の埋め込み kw ⊂ k(v̄) は付値環の包含 O(w) ⊂ Ov̄ から誘導され、σ̃ on k(v̄) は恒等（σ ∈ Iv̄/v の定義：R2）なので、その制限である kw 上の σ̃ も恒等。これは R2 の塔の関係「Iv̄/v の制限は下の惰性群に入る」の具体形。

7

red(R) = red(Qσ) − red(Q) = red(Q)σ̃ − red(Q) = red(Q) − red(Q) = Õ。

▸ 根拠　1つ目の等号：red は準同型（Step 4）。2つ目：Galois 両立性 N2-B（Step 6 の条件を満たす）。3つ目：σ̃ = id（Step 6）。本編の1行計算の完全展開がこれ。

8

kw の標数 p は m を割らない。

▸ 根拠　p | m と仮定すると v(m) ≥ v(p) ≥ 1（v は p の上の素点：w | v | p）となり v(m) = 0（v ∉ S）に矛盾。

9

R = O、すなわち Qσ = Q。

▸ 根拠　R ∈ E(M)[m] ⊂ E(Mw)[m]（R5）。R4(ii)（Step 8 で適用条件 p ∤ m を確認済み）より red は E(Mw)[m] 上単射。red(R) = Õ = red(O)（Step 7）なので R = O。

10

したがって Iv̄/v は x(Q), y(Q) を固定し、K(Q) ⊂ Tv̄。Q が任意だったので Step 1 の言い換えにより定理 (b) が従う。

▸ 根拠　Step 9 は任意の σ ∈ Iv̄/v について成立（Step 3 で σ は任意に固定した）。点の固定 ⇒ アフィン座標の固定（N0 の P3 の議論と同じ）。固定体の定義。

(b) ∎
Section 06

(a) L/K はアーベルで指数は m を割る

1

N1 の Step 18 の単射準同型 ρ : GL/K ↪ Hom(E(K)/mE(K), E[m]) を使う。

▸ 根拠　N1（この段階では E[m] ⊂ E(K) を仮定している）。

2

右辺は「値ごとの加法」でアーベル群をなし、全ての元 f が m·f = 0 を満たす。

▸ 根拠　値の群 E[m] がアーベルで指数 m だから（(m·f)(x) = [m](f(x)) = O）。

3

GL/K はアーベル群の部分群と同型なのでアーベル、かつ全ての元の位数が m を割る（指数 | m）。

▸ 根拠　単射準同型の像は部分群。部分群はアーベル性と指数条件を継承する。

(a) ∎
Section 07

まとめと、仮定の使用状況

この時点で L/K について分かったこと
  • アーベル拡大で、Galois 群の指数は m を割る（(a)──E[m] ⊂ E(K) を使う）
  • 有限集合 S の外で不分岐（(b)──E[m] ⊂ E(K) を使わない。この事実は第5部で Selmer 群の有限性にそのまま再利用される）
  • まだ有限次とは言っていない──それを言うのが N3 の仕事
? 局所が大域を縛る、の正確な意味

証明を1ステップずつ並べて見ると、「不分岐」という大域体論の結論は、 Step 7 のたった3つの等号──準同型性・Galois両立性・惰性の自明作用──と、 Step 9 の形式群からの単射性に完全に還元されています。 幾何（還元曲線）が見えなくする情報（Step 7）と、幾何が失わない情報（Step 9）の間に R = Qσ − Q が挟まれて消滅する。これが「良い還元が分岐を禁止する」の実体です。
