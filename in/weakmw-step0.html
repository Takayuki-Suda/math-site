Reduction Lemma, with every step justified

「E[m] ⊂ E(K) と仮定してよい」──本編 STEP 0 のこの一行を、飛躍ゼロで組み立て直します。 使う前提を先に番号付きで固定し、証明の各ステップに「主張」と「根拠」を明示します。 本編が地図なら、このノートは等高線まで描いた測量図です。

本編Ⅱ 弱MW N0 帰着補題 N1 Kummer対 N2 不分岐性 N3 体論の有限性
Section 01

このノートで埋める行間

本編 STEP 0 の証明は正しい設計図ですが、次の6箇所で読者の頭の中の補完に頼っていました。 このノートはそれを全部、紙の上に出します。

  • ①「自然な写像 E(K)/mE(K) → E(L)/mE(L)」が well-defined である理由
  • ②「有限群に挟まれて有限」という言い回しの集合論的に正確な形（第一同型定理）
  • ③ λP の構成に必要な2重の選択（剰余類の代表元と、そのm等分点）の明示
  • ④「Galois不変な点は K-有理」──点の相等と座標の相等の関係まで遡った正確な証明
  • ⑤ [m] と Galois 作用の可換性（証明中で暗黙に3回使われる）の根拠
  • ⑥ 系で使う「K(E[m])/K は有限次 Galois」の完全な証明

方針：まず Section 02 で前提を P1–P5 として固定します。以後の証明では、 すべてのステップが「P1–P5・既に示したステップ・純粋な計算」のいずれかだけから従うことを確認しながら進みます。

Section 02

前提の固定（P1–P5）

ここに挙げるものだけを外部から輸入します。それぞれ出典を明示します。 逆に言えば、以後の証明でこれ以外の「魔法」は一切使いません。

前提 P1 ─ 点集合の群構造 AEC III.2.2 / III.3.6

E(K) は O = [0:1:0] を単位元とするアーベル群であり、 加法と逆元は K 係数の射で与えられる。 中間体 K ⊂ F ⊂ K に対し E(F) は部分群である。 [m] : E(K) → E(K) は加法の反復なので群準同型である。

前提 P2 ─ Galois作用と加法の可換性

σ ∈ GK/K は P = (x, y) ↦ Pσ = (xσ, yσ)、Oσ = O によって E(K) に作用し、

(P + Q)σ = Pσ + Qσ,　(−P)σ = −Pσ,　したがって ([m]P)σ = [m](Pσ)

が成り立つ。また P ∈ E(K) ならば Pσ = P（座標が K の元で、σ は K を固定するから）。

なぜ可換か（証明を開く）

加法 P + Q の座標は、P, Q の座標の K 係数の有理式で書ける（弦と接線の公式：AEC III.2.3）。 σ は K を固定する体準同型なので、K 係数の有理式 f に対し f(x₁,…)σ = f(x₁σ,…) が成り立つ。 よって (P+Q)σ の座標 = 加法公式に σ を通したもの = Pσ + Qσ の座標。 場合分け（P = ±Q、O が絡む場合）も O = [0:1:0] が K-有理であることから同様。 ([m]P)σ = [m](Pσ) は (P+Q)σ = Pσ+Qσ を m−1 回適用するだけである。

前提 P3 ─ Galois不変な点は下の体の点

M/F を（有限次または無限次の）Galois 拡大とすると E(M)GM/F = E(F)。

証明を開く（④の行間）

⊇ は P2 の最後の主張。⊆ を示す。P ∈ E(M) が全ての σ で Pσ = P を満たすとする。 P = O なら O ∈ E(F) で終わり。P ≠ O なら P はアフィン座標 (x, y) を持ち、 アフィン座標は点を一意に表すので、「点として Pσ = P」から 「座標として xσ = x かつ yσ = y」が従う（ここが行間④：射影座標なら比の一致しか言えないが、 アフィン座標の組は一意である）。 よって x, y ∈ MGM/F = F。最後の等号は Galois 理論の基本定理 （無限次の場合も「Galois 拡大の固定体は基礎体」は成立：体論）。ゆえに P ∈ E(F)。

前提 P4 ─ m分点の個数 AEC III.6.2(d)

char K = 0 のとき E[m] := ker( [m] : E(K) → E(K) ) ≅ (ℤ/mℤ)²。 特に #E[m] = m² で、E[m] は E(K) の有限部分群。

前提 P5 ─ 群論の標準事実（第一同型定理）

φ : A → B を群準同型、N = ker φ とすると A/N ≅ im φ。 したがって A は N の剰余類 |im φ| 個に分割され、 N と im φ が共に有限なら |A| = |N|・|im φ| で A も有限。

Section 03

主張と証明の設計

補題（帰着補題） AEC VIII.1.1.1

L/K を有限次 Galois 拡大とする。E(L)/mE(L) が有限ならば E(K)/mE(K) も有限である。

設計：制限写像 E(K)/mE(K) → E(L)/mE(L) の核 Φ を、有限集合 Map(GL/K, E[m])（GL/K から E[m] への写像全体の集合）へ単射で送り込みます。 核と像が両方有限なら全体が有限（P5）、という2段構えです。

Section 04 ─ Part 1

核 Φ の特定と「挟み撃ち」の正確な形

1

mE(K) ⊆ mE(L)。したがって合成写像 E(K) ↪ E(L) ↠ E(L)/mE(L) は部分群 mE(K) を 0 に送る。

▸ 根拠　mE(K) の元は [m]R（R ∈ E(K)）の形で、E(K) ⊆ E(L)（P1）より R ∈ E(L)、よって [m]R ∈ mE(L)。

2

群準同型 φ : E(K)/mE(K) → E(L)/mE(L)、φ(P mod mE(K)) = P mod mE(L) が well-defined に誘導される。

▸ 根拠　商群の普遍性：核に mE(K) を含む準同型（Step 1）は商 E(K)/mE(K) を経由する。これが行間①の中身。

3

Φ := ker φ = ( E(K) ∩ mE(L) ) / mE(K)。

▸ 根拠　P mod mE(K) ∈ ker φ ⟺ P ∈ mE(L)（φ の定義）⟺ P ∈ E(K) ∩ mE(L)（P はもともと E(K) の元）。

4

E(L)/mE(L) が有限のとき、Φ が有限なら E(K)/mE(K) は有限で、|E(K)/mE(K)| = |Φ|・|im φ| ≤ |Φ|・|E(L)/mE(L)|。

▸ 根拠　P5 を A = E(K)/mE(K), N = Φ に適用。im φ は有限群 E(L)/mE(L) の部分集合なので有限。これが行間②の正確な形。以後、目標は「Φ は有限」だけになる。

Section 05 ─ Part 2

写像 λ の構成 ─ 2重の選択を明示する

5

各剰余類 c ∈ Φ に対し、代表元 Pc ∈ E(K) ∩ mE(L) を1つ選ぶ。さらに [m]Qc = Pc となる Qc ∈ E(L) を1つ選ぶ。この2つの選択を以後固定する。

▸ 根拠　代表元の存在は剰余類の定義。Qc の存在は Pc ∈ mE(L) の定義そのもの。これが行間③：以後の議論は「選んだ Pc, Qc」に依存した写像 c ↦ λc についてのものであり、それで論理は完結する（単射性さえ言えれば濃度の比較には十分）。

6

λc : GL/K → E(K) を λc(σ) := Qcσ − Qc で定義する。

▸ 根拠　Qc ∈ E(L) で GL/K は E(L) に作用する（P2 を L/K に適用）ので右辺は意味を持つ。

7

λc の値は実は E[m] に入る。すなわち λc ∈ Map(GL/K, E[m])。

▸ 根拠　次の3行の計算。各行の等号の理由を右に付す。

[m](Qcσ − Qc) = [m](Qcσ) − [m]Qc ← [m] は準同型（P1）
　= ([m]Qc)σ − [m]Qc ← 作用と [m] の可換性（P2）＝行間⑤
　= Pcσ − Pc = O ← Pc ∈ E(K) と P2 の最後の主張
Section 06 ─ Part 3

単射性と有限性

8

単射性：λc = λc′ ならば c = c′。以下 8a–8d に分解する。

8a

仮定より、すべての σ ∈ GL/K で Qcσ − Qc = Qc′σ − Qc′。

▸ 根拠　λc = λc′ は写像としての相等、つまり全ての引数での値の一致。

8b

移項して (Qc − Qc′)σ = Qc − Qc′（∀σ ∈ GL/K）。

▸ 根拠　E(K) はアーベル群（P1）なので移項は自由。(A − B)σ = Aσ − Bσ は P2。

8c

R := Qc − Qc′ とおくと R ∈ E(L) かつ GL/K 不変。よって R ∈ E(K)。

▸ 根拠　E(L) は部分群（P1）なので差も E(L) の点。不変性は 8b。結論は P3 を M = L, F = K に適用（L/K は Galois という仮定をここで使用）。

8d

[m]R = Pc − Pc′。R ∈ E(K) だから Pc − Pc′ ∈ mE(K)、すなわち c = c′。

▸ 根拠　[m]R = [m]Qc − [m]Qc′（P1：[m] は準同型）= Pc − Pc′（Step 5 の選択）。「c = c′」は商群 E(K)/mE(K) における相等の定義（差が mE(K) に入ること）。

9

Map(GL/K, E[m]) は有限集合で、その濃度はちょうど (m²)[L:K]。

▸ 根拠　|GL/K| = [L:K] < ∞（有限次 Galois 拡大：体論）、#E[m] = m²（P4）。有限集合 X から有限集合 Y への写像は |Y||X| 個。

10

c ↦ λc は Φ から Map(GL/K, E[m]) への単射（Step 8）。よって |Φ| ≤ (m²)[L:K] < ∞。Step 4 と合わせて補題の証明が完了する。

▸ 根拠　単射があれば濃度は値域の濃度以下（集合論）。

∎
副産物：明示的な上界
|E(K)/mE(K)| ≤ m2[L:K] ・ |E(L)/mE(L)|

「有限である」だけでなく定量的な評価が付いてくることに注意。 Mordell–Weil 階数の明示的上界（本編 STEP 3 の補足）は、この種の評価の積み重ねです。

何を使わなかったか（証明の輪郭を知る）

GL/K のアーベル性は不要（どんな有限次 Galois 拡大でもよい）。 L/K の有限性は Step 9 でのみ使った。 E[m] ⊂ E(K) の仮定は一切使っていない──だからこそ、この補題で 「E[m] を有理化する拡大」へ安全に登れます。

Section 07

系 ─ E[m] ⊂ E(K) と仮定してよい

系

L := K(E[m])（E[m] の全ての点のアフィン座標を K に添加した体）とおくと、 L/K は有限次 Galois 拡大である。したがって帰着補題より、 弱 Mordell–Weil 定理は E[m] ⊂ E(K) の場合に示せば十分である。

1

L/K は有限次拡大である。

▸ 根拠　E[m] は有限集合（P4：m² 個）で、各点の座標は K の元、つまり定義により K 上代数的。有限個の代数的元で生成される拡大は有限次（体論の標準事実）。「代数的か」は問題にすらならない──E(K) の点の座標は最初から代数的閉包の中にいる。これが行間⑥の前半。

2

GK/K は集合 E[m] を保つ：T ∈ E[m] ⇒ Tσ ∈ E[m]。

▸ 根拠　[m](Tσ) = ([m]T)σ（P2）= Oσ = O（P2：O は K-有理）。

3

L/K は正規、したがって（char 0 ゆえ分離的でもあり）Galois。

▸ 根拠　任意の K-埋め込み τ : L → K は K の自己同型 σ に延長できる（同型延長定理：体論）。Step 2 より σ は E[m] を置換するので、生成元の集合 {x(T), y(T) : T ∈ E[m]} を集合として保つ。よって τ(L) = K(σ(生成元)) = K(生成元) = L。「全ての埋め込みで像が L」は正規性の定義。

4

E(L)/mE(L) が有限だと示せば（E/L は E[m] ⊂ E(L) を満たす楕円曲線）、帰着補題により E(K)/mE(K) も有限。

▸ 根拠　Step 1–3 で L/K は有限次 Galois。E[m] ⊂ E(L) は L の作り方そのもの。E[m] は係数体の拡大で変わらない（K の中で定義された集合だから）。

∎
? λ は何者か ─ 第3部への橋

λc(σ) = Qcσ − Qc は、第3部の言葉では GL/K の E[m] における1-コサイクルです （E[m] への作用が自明とは限らないので、一般には準同型ではありません）。 今回「写像の集合 Map に単射で入る」という素朴な議論で済んだのは、濃度の評価しか要らなかったからです。 構造まで欲しくなったとき（どの λ が実現するか？）、コホモロジー群 H¹ が必要になります。
